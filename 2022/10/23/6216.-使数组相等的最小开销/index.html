дҪ еҸҜд»Ҙжү§иЎҢдёӢйқўж“ҚдҪң д»»ж„Ҹ ж¬Ўпјҡ

  • е°Ҷ nums дёӯ д»»ж„Ҹ е…ғзҙ еўһеҠ жҲ–иҖ…еҮҸе°Ҹ 1 гҖӮ

еҜ№з¬¬ i дёӘе…ғзҙ жү§иЎҢдёҖж¬Ўж“ҚдҪңзҡ„ејҖй”ҖжҳҜ cost[i] гҖӮ

иҜ·дҪ иҝ”еӣһдҪҝ nums дёӯжүҖжңүе…ғзҙ  зӣёзӯү зҡ„ жңҖе°‘ жҖ»ејҖй”ҖгҖӮ

зӨәдҫӢ 1пјҡ

иҫ“е…Ҙпјҡnums = [1,3,5,2], cost = [2,3,1,14]
иҫ“еҮәпјҡ8
и§ЈйҮҠпјҡжҲ‘д»¬еҸҜд»Ҙжү§иЎҢд»ҘдёӢж“ҚдҪңдҪҝжүҖжңүе…ғзҙ еҸҳдёә 2 пјҡ
- еўһеҠ з¬¬ 0 дёӘе…ғзҙ  1 ж¬ЎпјҢејҖй”Җдёә 2 гҖӮ
- еҮҸе°Ҹз¬¬ 1 дёӘе…ғзҙ  1 ж¬ЎпјҢејҖй”Җдёә 3 гҖӮ
- еҮҸе°Ҹз¬¬ 2 дёӘе…ғзҙ  3 ж¬ЎпјҢејҖй”Җдёә 1 + 1 + 1 = 3 гҖӮ
жҖ»ејҖй”Җдёә 2 + 3 + 3 = 8 гҖӮ
иҝҷжҳҜжңҖе°ҸејҖй”ҖгҖӮ

зӨәдҫӢ 2пјҡ

иҫ“е…Ҙпјҡnums = [2,2,2,2,2], cost = [4,2,8,1,3]
иҫ“еҮәпјҡ0
и§ЈйҮҠпјҡж•°з»„дёӯжүҖжңүе…ғзҙ е·Із»Ҹе…ЁйғЁзӣёзӯүпјҢдёҚйңҖиҰҒжү§иЎҢйўқеӨ–зҡ„ж“ҚдҪңгҖӮ

жҸҗзӨәпјҡ

  • n == nums.length == cost.length
  • 1 <= n <= 105
  • 1 <= nums[i], cost[i] <= 106

solution

еҸӮиҖғ

class Solution:
    def minCost(self, nums: List[int], cost: List[int]) -> int:
        d = list(zip(nums, cost))
        d = sorted(d)
        mid = None
        count = 0
        total = sum(cost)
        for a, b in d:
            count += b
            if count >= total // 2:
                mid = a
                break
        return sum(abs(a - mid) * c for a, c in d)
