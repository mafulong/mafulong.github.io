pythonйЎ№зӣ®зҺҜеўғ

2022/10/04 Python е…ұ 1155 еӯ—пјҢзәҰ 4 еҲҶй’ҹ

pythonйЎ№зӣ®зҺҜеўғ

pip

еҸӮиҖғ

д»ҘдёӢжҳҜеҜ№зі»з»ҹpythonзҺҜеўғдҝ®ж”№пјҢеҰӮжһңжҳҜйЎ№зӣ®зә§еҲ«зҡ„пјҢи®°дҪҸе…ҲvenvеҗҺеҶҚvenvйҮҢиҝӣиЎҢpip3зҡ„ж“ҚдҪң

дҪ еҸҜд»ҘйҖҡиҝҮд»ҘдёӢе‘Ҫд»ӨжқҘеҲӨж–ӯжҳҜеҗҰе·Іе®үиЈ…пјҡ

pip3 --version    # Python3.x зүҲжң¬е‘Ҫд»Ө
# еҸҜд»Ҙalias pip=pip3
pip install -U pip # pipеҚҮзә§

жҢҮе®ҡpipд№ҹеҸҜд»Ҙ

sudo pip2 install жЁЎеқ—еҗҚ

pipжәҗдҝ®ж”№ йҳҝйҮҢжәҗеҚіеҸҜ

# жё…еҚҺжәҗ
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple

# жҲ–пјҡ
# йҳҝйҮҢжәҗ
pip config set global.index-url https://mirrors.aliyun.com/pypi/simple/
# и…ҫи®Ҝжәҗ
pip config set global.index-url http://mirrors.cloud.tencent.com/pypi/simple
# иұҶз“Јжәҗ
pip config set global.index-url http://pypi.douban.com/simple/

requirements

йҖӮз”ЁдәҺ еҚ•иҷҡжӢҹзҺҜеўғзҡ„жғ…еҶөпјҡдёәд»Җд№ҲеҸӘйҖӮз”ЁдәҺеҚ•иҷҡжӢҹзҺҜеўғпјҹеӣ дёәиҝҷз§Қж–№ејҸпјҢдјҡе°ҶзҺҜеўғдёӯзҡ„дҫқиө–еҢ…е…ЁйғҪеҠ е…ҘпјҢеҰӮжһңдҪҝз”Ёзҡ„е…ЁеұҖзҺҜеўғпјҢеҲҷдёӢиҪҪзҡ„жүҖжңүеҢ…йғҪдјҡеңЁйҮҢйқўпјҢдёҚз®ЎжҳҜдёҚж—¶еҪ“еүҚйЎ№зӣ®дҫқиө–зҡ„

йқһvenvиҜ·дёҚиҰҒз”ЁиҝҷдёӘ

жҹҘзңӢеҪ“еүҚpip installзҡ„

pip3 list # иҫ“еҮәеҪ“еүҚеҲ—иЎЁ

жӣҙж–°requirementsж–Үд»¶пјҢжҜҸж¬Ўpip installж–°зҡ„еҗҺйғҪйңҖиҰҒжү§иЎҢдёҖж¬Ў

pip freeze > requirements.txt

дҪҝз”Ёrequirements.txtе®үиЈ…дҫқиө–

pip install ``-``r requirements.txt

venv иҷҡжӢҹзҺҜеўғ

еҸӮиҖғ

иҷҡжӢҹзҺҜеўғпјҢе’Ңзі»з»ҹPythonзҺҜеўғйҡ”зҰ»пјҢеҘҪеӨ„жҳҜдә’дёҚеҪұе“ҚпјҢйЎ№зӣ®зә§еҲ«жҺЁиҚҗгҖӮ

еҲӣе»әиҷҡжӢҹзҺҜеўғ (еҗҺйқўvenvжҳҜеҗҚеӯ—пјҢдёҖдёӘйЎ№зӣ®еҸҜд»ҘеӨҡдёӘvenvпјҢиҝҷйҮҢе°ұеҸ«venvпјҢдёҚз”Ёиө·е…¶д»–еҗҚеӯ—дәҶ)

python3 -m venv venv

иҰҒдҪҝз”ЁиҷҡжӢҹзҺҜеўғе°ұеҝ…йЎ»жҝҖжҙ»

 вһң source venv/bin/activate

з„¶еҗҺеңЁе‘Ҫд»ӨиЎҢдёӯе°ұдјҡеҮәзҺ°

(venv) вһң

иҜҙжҳҺиҷҡжӢҹзҺҜеўғе·Із»ҸжҝҖжҙ»пјҢиҝӣе…ҘеҲ°venv shellдәҶ

venvйҮҢжңүpipе’ҢPip3жҸҗдҫӣ, python3д№ҹжҳҜvenvиҮӘе·ұзҡ„пјҢзӣёеҪ“дәҺдёҖдёӘж–°зҡ„shellпјҢз„¶еҗҺиҰҶзӣ–дәҶдёҖдәӣpythonзӣёе…іе‘Ҫд»Ө

иҰҒеҸ–ж¶ҲиҷҡжӢҹзҺҜеўғпјҢдҪҝз”Ё

(venv) вһң deactivate

pycharmз»“еҗҲдҪҝз”Ё

еҸҜйҖҡиҝҮvenvе‘Ҫд»ӨиЎҢеҲӣе»әеҘҪpythonзҺҜеўғпјҢз„¶еҗҺдҪҝз”Ёpycharmи®ҫзҪ®йҮҢи®ҫзҪ®interpreter жқҘеҲҮжҚўеҲ°venvгҖӮ

example

  1. еҲӣе»әvenv, еҰӮжһңжңүе°ұжҝҖжҙ»
  2. venv shellйҮҢpip requirementж“ҚдҪңпјҢе®үиЈ…е’Ңжӣҙж–°requirementж–Үд»¶
  3. pycharmи®ҫзҪ®interpreter
