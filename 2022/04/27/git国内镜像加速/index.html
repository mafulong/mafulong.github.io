git 国内镜像加速

安装 cgit

cgit 可直接代替 git 命令，cgit 内部会自动把 github.com 域名替换成想要的国内镜像域名。

安装 cgit

想要的国内镜像域名可以通过 bash_profile 里加如下配置

export CGIT_MIRROR=https://hub.xn--gzu630h.xn--kpry57d/

然后把脚本里的 git 改成 cgit，或者 alias 直接改 git -> cgit
