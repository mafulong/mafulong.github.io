## title

操作给定的二叉树，将其变换为源二叉树的镜像。

	二叉树的镜像定义：源二叉树 
	    	    8
	    	   /  \
	    	  6   10
	    	 / \  / \
	    	5  7 9 11
	    	镜像二叉树
	    	    8
	    	   /  \
	    	  10   6
	    	 / \  / \
	    	11 9 7  5

## solution


```java
public class Solution {
    public void Mirror(TreeNode root) {
        if(root==null)
            return;
        Mirror(root.left);
        Mirror(root.right);

        TreeNode t=root.right;
        root.right=root.left;
        root.left=t;

    }
}

```
