# qt打开网页

## 方法一 使用默认浏览器

1. 需要包含的头文件有
```c++
    #include <QDesktopServices>
    #include <QUrl> 
```

2. 代码中打开网页
```c++
    QDesktopServices::openUrl(QUrl(QString("www.baidu.com")));
    QUrl url(QString("www.baidu.com"));
   //可选
    bool is_open = QDesktopServices::openUrl(url);
```
