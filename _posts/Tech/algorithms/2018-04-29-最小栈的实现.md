title: 最小栈的实现
---

一个栈保存正常数据，一个栈保存当前栈中的最小值，也就是改下push和pop，记得要更新当前栈中的最小值撒

最大栈和这个原理都一样的

```c++
class MinStack {
public:
	/** initialize your data structure here. */
	int min1;
	stack<int> a, b;
	MinStack() {
		min1 = INT_MAX;
	}

	void push(int x) {
		min1 = min(min1, x);
		a.push(x);
		b.push(min1);
	}

	void pop() {
		a.pop();
		b.pop();
		if (b.empty())
			min1 = INT_MAX;
		else
			min1 = b.top();
	}

	int top() {
		return a.top();
	}

	int getMin() {
		return b.top();
	}
};
```
