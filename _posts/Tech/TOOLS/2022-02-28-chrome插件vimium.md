## [使用 chrome 扩展 Vimium 实现快捷键关闭其他标签页](https://www.cnblogs.com/36bian/p/7669594.html)

自定义配置

```
# <<和>>默认是左移右移标签，用的少我们覆盖掉好了
map << closeTabsOnLeft
map >> closeTabsOnRight
map cc closeOtherTabs
```
