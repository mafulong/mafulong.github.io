## win11开机无密码

[参考](https://support.lenovo.com/ie/en/solutions/ht503518-how-to-remove-login-password-from-windows-10)

## 方法一：netplwiz（最简单）

### 1. 打开用户账户设置

按 `Win + R`，输入 `netplwiz`，点击确定。

### 2. 取消登录要求

在用户列表中选中当前账户，取消勾选「要使用本计算机，用户必须输入用户名和密码」，点击「应用」。

### 3. 输入密码确认

系统会要求输入当前用户名和密码，输入后确认即可。

### 4. 重启验证

重启后应自动进入桌面。

### 如果看不到「用户必须输入用户名和密码」选项

微软账户启用了 Windows Hello 时，经常会隐藏这个选项。关闭 Windows Hello：

`设置` → `账户` → `登录选项` → 关闭「为了提高安全性，仅允许对此设备上的 Microsoft 帐户使用 Windows Hello 登录」

然后重新运行 `netplwiz`，选项通常就会出现。

## 方法二：PowerShell 注册表（适合远程操作）

管理员 PowerShell：

```powershell
Set-ItemProperty "HKLM:\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon" "AutoAdminLogon" -Value "1"
Set-ItemProperty "HKLM:\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon" "DefaultUserName" -Value "用户名"
Set-ItemProperty "HKLM:\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon" "DefaultPassword" -Value "密码"
```
