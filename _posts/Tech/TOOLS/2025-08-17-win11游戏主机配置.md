### 游戏模式

1. **开启游戏模式**
   - 打开 *设置 → 游戏 → 游戏模式*，确保开启。
2. **开启硬件加速 GPU 计划**
   - 设置 → 显示 → 图形 → 更改默认图形设置 → 开启“硬件加速 GPU 计划”。
3. **Steam 玩家注意事项**
   - 打开 Steam → 好友 → 离线模式。
   - 避免后台好友列表占用，帧数会更稳定。
4. **独显直连**
   - 确保开启独显直连（特别是笔记本用户）。
5. **电源模式**
   - 联想用户：超能模式 < 野兽模式 → 建议开启“野兽模式”。
6. **网络加速**
   - 推荐使用加速器（如雷神），能降低延迟。



**系统设置优化**

1. **显卡控制面板 → 调整为性能模式**
2. **显存不足时 → 开启内存回退**
3. **电源管理 → 最高性能优先**
4. **系统 → 屏幕 → 图形 → 开启 GPU 硬件加速 → 重启电脑**





## 显示器设置

小米电视gpro27u



**关闭自动亮度**

- 默认开启 → 会错误调整伽马和亮度，导致观感不好。



**游戏模式优化**

- **进入方式**：遥控器 → 长按“三条杠” → 二级菜单 → 设置。
- **推荐参数**：
  - 亮度：40（默认 70，过高伤眼）
  - 色温：改为 **标准**（避免暖色 / 偏红）
  - 色彩增强：关闭
  - 色彩空间：P3
  - FreeSync：开启（防撕裂）
  - 响应时间：普通（快速模式会过冲）. 

📌 调整后：画面不再发红，色彩自然，适合日常与游戏使用。



不建议在 NVIDIA 控制面板里修改数字通道：

- HDR 下可能有效，但会影响 SDR 下的色准。
- 尽量在显示器硬件里直接调节。



**暗部增强功能**

- 提升暗部细节辨识度（例如黑夜地图能更容易发现敌人）。
- 对 FPS 玩家非常实用。



# 游戏

## 软件设置 Cod

**显示设置**

- **显示模式**：全屏专用
- **显示器**：选择主显示器
- **显卡**：确保使用独显，不要用集显
- **刷新率**：与屏幕一致
- **分辨率**：根据自己屏幕设置
- **垂直同步**：关闭



**关闭语音聊天**

- 路径：音频设置 → 语音聊天 → 全关（推荐）。
- 效果：从 130 FPS → 160–200 FPS，大幅提升帧数，掉帧问题明显改善。
- 原因：语音频道后台占用系统资源，影响帧数。COD19、20 也有类似问题，但 COD21 更严重。



**DLSS / 帧生成**

- **DLSS / DLSSR**：建议开启（提升画质和性能）。
- **帧生成（Frame Generation）**：不建议开启。
  - 开启后帧数可冲到 200+ FPS，但输入延迟明显增加（从 <20ms → 40ms）。
  - FPS 游戏对延迟敏感 → 关掉更合适。
  - 单机 / 画面党可开启，但多人 FPS 建议关闭。
  - **注意事项**：需要在系统里开启 **GPU 硬件加速** 才能生效。
  - **提示**：开启后帧数更高，但可能带来输入延迟 → FPS 游戏需谨慎。



**显卡驱动务必保持最新**：

- 每次新作发布，官方都会更新驱动优化性能。
- NVIDIA → GeForce Experience 可一键更新。
- AMD 用户去官网输入显卡型号下载即可。



**锁定画面设置（视觉增强）**

1. **路径**：设置 → 界面 → 易读性 → 定制颜色 → 打开。

2. **敌方颜色饱和度 / 亮度拉满**。

3. **滤镜**：选择滤镜 2，两个颜色强度拉满。

4. **效果**：敌人更明显，画面更干净，提升发现敌人的效率。

   - 缺点：色调可能部分玩家不喜欢，可随时改回默认。

   

**质量设置**

- **渲染分辨率**：100（不要降低，否则会模糊）
- **动态分辨率**：关闭（帧数不足时可以开，但可能模糊/撕裂）
- **图像锐化**：
  - 推荐 **CAS**（75 左右），画质比大力水手更清晰
  - 大力水手帧数高，但噪点和模糊严重
- **超分辨率生成**：关闭
- **显存比例**：默认 80（显存不足可开到 90）
- **着色器预缓存**：一定要关闭，否则会频繁卡顿





 **画质与帧数测试结论**

作者花费 7–8 小时测试，逐项调整后得到：

- **对帧数影响最大**：

  - **子弹撞击效果**：关闭后可提升 ~30 帧。
  - **屏幕反射**：关闭或调低可提升 ~10–20 帧。

- **对帧数影响中等**：

  - **环境**：影响适中，需根据需求调整。

- **对帧数影响小但画质影响大**：

  - **全局质量、阴影和光照**：画质显著提升，但帧数损耗小。

- **材质分辨率**：必须设高，对画质影响大，对帧数影响小。

- **景深**：

  - 开启 → 开镜后枪模糊
  - 关闭 → 枪更清晰（推荐关闭）。

  

------

**材质与细节**



- **细节质量（石头草木等）**：低（性能优化明显）
- **网格解析度**：高（对帧率影响小，但能提升视觉和武器贴图特效）
- **粒子效果**：低（高会掉 10-20 帧）

------

**材质加载与传输**

- **高清材质流**：开优化（对画质有帮助，显存占用适中）
- **本地材质传输**：低（对战区大地图远景才有用，一般影响不大，显存占用高）





**其他提示**

- **锁帧设置**：COD20 有效，但本作无明显作用。
- **DLSS Bug**：有时切换后会自动变成 CAS → 需留意。
- **数据监控**：界面 → 状态 → 选择显示 FPS、延迟、显存占用等信息。



------

**注意事项**

- 每次修改画质设置后，**一定要重新预载着色器**：
   进入显示界面 → 预载 → 保存 → 退出游戏 → 重启再进游戏





## 软件设置 Apex

###  关键参数解释与建议

> 以“稳定帧率优先”为导向；如需更好画质可在稳定后微调。

① 各向异性过滤（Anisotropic Filtering）

- **影响**：主要吃显卡，**对帧数影响仅几帧**。
- **建议**：在中低端卡上也能开到 **8×**（示例：**GTX 1050 Ti @ 8×** 运行良好）。
- **取舍**：更高带来更清晰的斜向纹理，性价比高。

② 碰撞痕迹/冲击痕迹（Decals / Impact Marks）

- **影响**：**吃 CPU**。
- **建议**：**调为 0**，明显减负。

③ 纹理串流预算（Texture Streaming Budget）

- **原理**：与**显存大小**绑定，决定可加载纹理体量与切换速度。
- **操作**：
  1. 先在游戏里选择接近你显存的档位，**查看/记下实际“流内存大小”**；
  2. 回到配置文件写入相应值；
  3. **特殊规则**：如果你想设为 **2GB 显存**，配置文件中**该字段需写“1”**；**其余情况写“0”**（按原作者说明）。
- **策略**：在**不爆显存**前提下尽量拉高，提升清晰度与加载体验。

④ 模型刷新/细节（Model LOD/Detail）

- **建议**：**0.6**。
- **原因**：再低会让**中距离对枪“果冻/方块化”**（人物像块状），影响观察。

⑤ 阴影 & 环境光遮蔽（Shadows / Ambient Occlusion）

- **建议**：**全部关闭**。
- **原因**：这两项对视觉收益有限、对性能影响大。

⑥ 自适应分辨率 / 帧率目标（Dynamic Resolution / Adaptive）

- **定位**：本方案的**关键项**。
- **机制**：为保持**目标帧数**，在负载高时**动态降低分辨率/画质**，负载回落再提升。
- **用法**：
  - **启用**自适应（配置项打开），设置**目标帧率**（如 60/75/90/100）。
  - 分辨率基准建议 **1080p**。
  - **抗锯齿**（TAA 等）建议**开启**，与自适应配合可减轻低分辨率时的锯齿与闪烁。
- **实测参考（GTX 1050 Ti）**：
  - 启用自适应时，**上限约 ~100FPS**；
  - 若 100FPS 仍不满足，可将**与自适应相关的 3 项置 0**（等同关闭/限制），实际读数出现 **100–144FPS** 的波动；
  - **代价**：中距离识别变差（“红点看缘分”），画面锐度随负载波动较明显。
  - 建议：**先求稳再求高**，优先稳定的目标帧（如 90/100），再考虑进一步抬上限。

⑦ 抗锯齿（AA）

- **建议**：本方案里**默认开启**，与自适应联动更顺眼。
- **说明**：如极度追求清晰边缘与响应，可在稳定后尝试调低或关闭，但可能出现闪烁。
