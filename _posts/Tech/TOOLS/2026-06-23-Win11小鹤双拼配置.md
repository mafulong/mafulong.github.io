## 小鹤双拼配置

本文介绍如何通过 PowerShell 脚本将小鹤双拼导入微软拼音，并设为默认双拼方案。

## 原理

微软拼音支持用户自定义双拼方案，核心是注册表中的三个键值：

| 键名 | 类型 | 说明 |
|------|------|------|
| `Enable Double Pinyin` | DWord | 1 = 开启双拼 |
| `DoublePinyinScheme` | DWord | 10 = 使用用户自定义方案 |
| `UserDefinedDoublePinyinScheme0` | String | 小鹤双拼映射表 |

其中 `DoublePinyinScheme = 10` 是微软拼音为用户自定义方案预留的编号。

## 脚本

以管理员权限运行 PowerShell，执行以下命令：

```powershell
$Path = "HKCU:\Software\Microsoft\InputMethod\Settings\CHS"

New-Item -Path $Path -Force | Out-Null

Set-ItemProperty -Path $Path -Name "Enable Double Pinyin" -Type DWord -Value 1

Set-ItemProperty -Path $Path -Name "DoublePinyinScheme" -Type DWord -Value 10

Set-ItemProperty -Path $Path -Name "UserDefinedDoublePinyinScheme0" -Type String -Value "小鹤双拼*2*^*iuvdjhcwfg^xmlnpbksqszxkrltvyovt"

Write-Host "小鹤双拼已导入，请注销或重启后生效。"
```

## 验证

重启或注销后，按以下路径检查：

`设置` → `时间和语言` → `语言和区域` → `中文(简体，中国)` → `微软拼音` → `键盘选项` → `双拼`

应该能看到「小鹤双拼」选项并使用。

## 如未生效（Win11 24H2）

部分情况下设置界面仍不显示，可尝试重启输入法进程：

```powershell
Stop-Process -Name ctfmon -Force -ErrorAction SilentlyContinue
Start-Process ctfmon.exe
```

## 附：其他双拼方案对照

如果你想切换到其他双拼方案，只需修改 `DoublePinyinScheme` 的值和映射表字符串即可。常见的微软自带方案编号如下：

| 值 | 方案 |
|----|------|
| 0 | 微软拼音 ABC |
| 1 | 微软拼音智能 ABC |
| 2 | 微软双拼 |
| ... | ... |
| 10 | 用户自定义 |

具体各方案对应的映射表可参考小鹤官网或开源项目。
