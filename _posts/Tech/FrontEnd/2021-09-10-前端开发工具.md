## 前端开发工具

# vscode

## 必备插件

#### Auto Close Tag

自动闭合 HTML/XML 标签

### Auto Rename Tag

自动重命名 HTML/XML 标签

### Bracket Pair Colorizer

用于着色匹配括号，让你的代码分层更清晰

### Code Runner

运行代码片段或多种语言的代码文件

### Code Spell Checker

源代码拼写检查器，提示代码中单词拼写错误

### CSS Peek

允许查看css，并从HTML文件定位到css文件，文件定义跳转
