## 离散化



Sorted(Range(N))的用法。 



[参考题目](https://leetcode-cn.com/problems/odd-even-jump/)



```
B = sorted(range(N), key = lambda i: A[i])
```



比如A=[5,3,4], B = [1,2,0]， B存储的是A中index的递增排序，排序规则为A[i]大小


作用：

- 利用B[i]可以求出有序后位置排名第i位的原来位置。B[0]代表了有序后最小的位置，也就是1，A[1]是有序后最小的数字。
