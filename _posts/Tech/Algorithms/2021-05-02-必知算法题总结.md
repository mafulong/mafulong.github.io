- [判断一个整数是否是回文数](https://leetcode-cn.com/problems/palindrome-number/description/)  不能用额外存储空间

  - ```java
    public boolean isPalindrome(int x) {
        if (x == 0) {
            return true;
        }
        if (x < 0 || x % 10 == 0) {
            return false;
        }
        int right = 0;
        while (x > right) {
            right = right * 10 + x % 10;
            x /= 10;
        }
        return x == right || x == right / 10;
    }
    ```
