## github codecov

### 什么是Codecov？

[Codecov](https://codecov.io/gh)是一个测试结果分析工具，travis负责执行测试，Codecov负责分析测试结果，最简单的用法就是衡量测试代码覆盖度，当然更高端的用法还有待继续学习。依赖于travis，Codecov非常简单就能上手。



入门教程: https://cloud.tencent.com/developer/article/1433286
