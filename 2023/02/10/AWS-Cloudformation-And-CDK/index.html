AWS Cloud Development Kit

The AWS Cloud Development Kit (CDK) provides some of the same benefits of CloudFormation but with a few key differences.

The CDK is an infrastructure-as-code solution that you can use with several popular programming languages. In other words, itвАЩs like CloudFormation, but using a language you already know. The CDK also contains command line tools to create infrastructure-as-code templates and to instantiate, update, and tear down stacks.
