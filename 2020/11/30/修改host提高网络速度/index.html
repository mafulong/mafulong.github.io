дҝ®ж”№hostжҸҗй«ҳзҪ‘з»ңйҖҹеәҰ

2020/11/30 Tools е…ұ 1305 еӯ—пјҢзәҰ 4 еҲҶй’ҹ

Hosts зҡ„жҰӮеҝө

Hosts жҳҜдёҖдёӘжІЎжңүжү©еұ•еҗҚзҡ„зі»з»ҹж–Үд»¶пјҢеҸҜд»Ҙз”Ёи®°дәӢжң¬зӯүе·Ҙе…·жү“ејҖпјҢе…¶дҪңз”Ёе°ұжҳҜе°ҶдёҖдәӣеёёз”Ёзҡ„зҪ‘еқҖеҹҹеҗҚдёҺе…¶еҜ№еә”зҡ„ IP ең°еқҖе»әз«ӢдёҖдёӘе…іиҒ”вҖңж•°жҚ®еә“вҖқпјҢеҪ“з”ЁжҲ·еңЁжөҸи§ҲеҷЁдёӯиҫ“е…ҘдёҖдёӘйңҖиҰҒзҷ»еҪ•зҡ„зҪ‘еқҖж—¶пјҢзі»з»ҹдјҡйҰ–е…ҲиҮӘеҠЁд»Һ Hosts ж–Үд»¶дёӯеҜ»жүҫеҜ№еә”зҡ„ IP ең°еқҖпјҢдёҖж—ҰжүҫеҲ°пјҢзі»з»ҹдјҡз«ӢеҚіжү“ејҖеҜ№еә”зҪ‘йЎөпјҢеҰӮжһңжІЎжңүжүҫеҲ°пјҢеҲҷзі»з»ҹдјҡеҶҚе°ҶзҪ‘еқҖжҸҗдәӨ DNS еҹҹеҗҚи§ЈжһҗжңҚеҠЎеҷЁиҝӣиЎҢ IP ең°еқҖзҡ„и§ЈжһҗгҖӮHosts зҡ„иҜ·жұӮзә§еҲ«жҜ” DNS й«ҳгҖӮ

Hosts ж–Үд»¶ж јејҸ

127.0.0.1 localhost 127.0.0.1 http://www.baidu.com 192.168.1.2 http://test.cn

еҹҹеҗҚи§ЈжһҗжөҒзЁӢпјҡ

ж №жҚ®д»ҘдёҠжөҒзЁӢпјҢжҲ‘д»¬е°ұеҸҜд»Ҙд»Һ hosts ж–Үд»¶дёҠеҒҡжүӢи„ҡпјҢе°Ҷе№ҝе‘Ҡзҡ„зӣёе…іеҹҹеҗҚи§Јжһҗдёә 127.0.0.1пјҢиҝҷдёӘең°еқҖжҳҜжң¬жңәеӣһйҖҒең°еқҖпјҲLoopback AddressпјүпјҢдё»иҰҒз”ЁдәҺзҪ‘з»ңиҪҜд»¶жөӢиҜ•д»ҘеҸҠжң¬ең°жңәиҝӣзЁӢй—ҙйҖҡдҝЎпјҢж— и®әд»Җд№ҲзЁӢеәҸпјҢдёҖж—ҰдҪҝз”ЁеӣһйҖҒең°еқҖеҸ‘йҖҒж•°жҚ®пјҢеҚҸи®®иҪҜд»¶з«ӢеҚіиҝ”еӣһпјҢдёҚиҝӣиЎҢд»»дҪ•зҪ‘з»ңдј иҫ“ гҖӮд№ҹе°ұдҪҝз”ЁдәҶ 127.0.0.1 е№ҝе‘ҠеҹҹеҗҚ зҡ„й…ҚзҪ® hosts ж–Үд»¶пјҢиҜ·жұӮе№ҝе‘Ҡзҡ„зҪ‘з»ңиҜ·жұӮеҸ‘йҖҒпјҢд»ҺиҖҢдёҚиғҪеұ•зӨәе№ҝе‘ҠпјҢиҝҷж ·е°ұиҫҫеҲ°еҺ»е№ҝе‘Ҡзҡ„зӣ®зҡ„гҖӮе…ідәҺеҺ»е№ҝе‘Ҡж–№жЎҲз•ҷеңЁдёӢзҜҮеұ•ејҖгҖӮ

йҷӨдәҶеҺ»е№ҝе‘Ҡзҡ„еә”з”ЁпјҢиҝҳеә”з”ЁеңЁз§‘еӯҰдёҠзҪ‘гҖҒж–ӯејҖзү№е®ҡиҝһжҺҘгҖӮдҝ®ж”№ hosts ж–Үд»¶е°ұеҸҜд»ҘйҒҝе…Қ DNS жұЎжҹ“гҖӮз»ҸеёёзңӢдёҖдёӢз ҙи§ЈжҸҗзӨәе°ұжҳҜйңҖиҰҒеңЁ hosts ж–Үд»¶ж·»еҠ  127.0.0.1 еҹҹеҗҚжқҘдҪҝиҪҜд»¶ж— жі•иҝһжҺҘзҪ‘з»ңгҖӮ

дҝ®ж”№ hosts ж–Үд»¶

жҺҘдёӢжқҘе…ій”®й—®йўҳе°ұжҳҜеҰӮдҪ•дҝ®ж”№ hosts ж–Үд»¶пјҢhosts ж–Үд»¶еӯҳж”ҫзӣ®еҪ•пјҡ

  • Window зӣ®еҪ• C:WindowsSystem32driversetc
  • Linux еҸҠе…¶д»–зұ» Unix ж“ҚдҪңзі»з»ҹпјҡ/etc
  • Android зӣ®еҪ•/system/etc/

зј–иҫ‘дҝ®ж”№д»ҘдёҠзӣ®еҪ•дёӢзҡ„ hosts ж–Үд»¶гҖӮзӣ®еүҚе®һзҺ°ж–№ејҸжңүи®ёеӨҡеҪўејҸпјҢжңҖзӣҙжҺҘзҡ„е°ұжҳҜдҪҝз”Ёж–Үд»¶з®ЎзҗҶеҷЁе°ҶиҺ·еҸ–зҡ„ hosts ж–Үд»¶иҰҶзӣ–зі»з»ҹзҡ„ hosts ж–Үд»¶гҖӮ

еҲ·ж–° DNS зј“еӯҳ

гҖҗWindowsгҖ‘ жҢүдёӢ Windows R й”®пјҢиҝҗиЎҢ cmd пјҢеңЁе‘Ҫд»ӨжҸҗзӨәз¬ҰиҝҗиЎҢе‘Ҫд»Ө ipconfig /flushdns

гҖҗOS X 10.9гҖ‘гҖҗOS X 10.10.4 гҖ‘еңЁпј»еә”з”ЁзЁӢеәҸпјҪпј»е®һз”Ёе·Ҙе…·пјҪпј»з»Ҳз«ҜпјҪиҝҗиЎҢе‘Ҫд»Ө sudo dscacheutil -flushcache; sudo killall -HUP mDNSResponder

гҖҗOS X 10.10гҖ‘ еңЁпј»еә”з”ЁзЁӢеәҸпјҪпј»е®һз”Ёе·Ҙе…·пјҪпј»з»Ҳз«ҜпјҪиҝҗиЎҢе‘Ҫд»Ө sudo discoveryutil udnsflushcaches

гҖҗOS X 10.7 ~ 10.8гҖ‘ еңЁпј»еә”з”ЁзЁӢеәҸпјҪпј»е®һз”Ёе·Ҙе…·пјҪпј»з»Ҳз«ҜпјҪиҝҗиЎҢе‘Ҫд»Ө sudo killall -HUP mDNSResponder

гҖҗLinuxгҖ‘ еңЁпј»з»Ҳз«ҜпјҪиҝҗиЎҢе‘Ҫд»Ө /etc/rc.d/init.d/nscd restart

гҖҗAndroidгҖҒiOSгҖ‘ йҮҚж–°еҗҜеҠЁи®ҫеӨҮ
