Title

the problem link

жҠҠеҸӘеҢ…еҗ«еӣ еӯҗ2гҖҒ3е’Ң5зҡ„ж•°з§°дҪңдё‘ж•°пјҲUgly NumberпјүгҖӮдҫӢеҰӮ6гҖҒ8йғҪжҳҜдё‘ж•°пјҢдҪҶ14дёҚжҳҜпјҢеӣ дёәе®ғеҢ…еҗ«еӣ еӯҗ7гҖӮ д№ жғҜдёҠжҲ‘д»¬жҠҠ1еҪ“еҒҡжҳҜз¬¬дёҖдёӘдё‘ж•°гҖӮжұӮжҢүд»Һе°ҸеҲ°еӨ§зҡ„йЎәеәҸзҡ„з¬¬NдёӘдё‘ж•°гҖӮ

Solution

иҜҙдёӢжҖқи·ҜпјҢеҰӮжһңpжҳҜдё‘ж•°пјҢйӮЈд№Ҳp=2^x * 3^y * 5^z йӮЈд№ҲеҸӘиҰҒиөӢдәҲx,y,zдёҚеҗҢзҡ„еҖје°ұиғҪеҫ—еҲ°дёҚеҗҢзҡ„дё‘ж•°гҖӮ еҰӮжһңиҰҒйЎәеәҸжүҫеҮәдё‘ж•°пјҢиҰҒзҹҘйҒ“дёӢйқўеҮ дёӘзү№пјҲfeiпјүзӮ№пјҲhuaпјүгҖӮ еҜ№дәҺд»»дҪ•дё‘ж•°pпјҡ

пјҲдёҖпјүйӮЈд№Ҳ2*p,3*p,5*pйғҪжҳҜдё‘ж•°пјҢе№¶дё”2*p<3*p<5*p
пјҲдәҢпјүеҰӮжһңp<q, йӮЈд№Ҳ2*p<2*q,3*p<3*q,5*p<5*q

зҺ°еңЁиҜҙиҜҙз®—жі•жҖқжғіпјҡ В В В В з”ұдәҺ1жҳҜжңҖе°Ҹзҡ„дё‘ж•°пјҢйӮЈд№Ҳд»Һ1ејҖе§ӢпјҢжҠҠ21пјҢ31пјҢ51пјҢиҝӣиЎҢжҜ”иҫғпјҢеҫ—еҮәжңҖе°Ҹзҡ„е°ұжҳҜ1 зҡ„дёӢдёҖдёӘдё‘ж•°пјҢд№ҹе°ұжҳҜ21пјҢ В В В В иҝҷдёӘж—¶еҖҷпјҢеӨҡдәҶдёҖдёӘдё‘ж•°вҖҳ2вҖҷпјҢд№ҹе°ұеҸҲеӨҡдәҶ3дёӘеҸҜд»ҘжҜ”иҫғзҡ„дё‘ж•°пјҢ22пјҢ32пјҢ52пјҢ иҝҷдёӘж—¶еҖҷе°ұжҠҠд№ӢеүҚвҖҳ1вҖҷз”ҹжҲҗзҡ„дё‘ж•°е’ҢвҖҳ2вҖҷз”ҹжҲҗзҡ„дё‘ж•°еҠ иҝӣжқҘд№ҹе°ұжҳҜ (31,51,22пјҢ32пјҢ52)иҝӣиЎҢжҜ”иҫғпјҢжүҫеҮәжңҖе°Ҹзҡ„гҖӮгҖӮгҖӮгҖӮеҰӮжӯӨеҫӘзҺҜдёӢеҺ»е°ұдјҡеҸ‘зҺ°пјҢ жҜҸж¬ЎйҖүиҝӣжқҘдёҖдёӘдё‘ж•°пјҢиҜҘдё‘ж•°еҸҲдјҡз”ҹжҲҗ3дёӘж–°зҡ„дё‘ж•°иҝӣиЎҢжҜ”иҫғгҖӮ В В В В дёҠйқўзҡ„жҡҙеҠӣж–№жі•д№ҹеә”иҜҘиғҪи§ЈеҶіпјҢдҪҶжҳҜеҰӮжһңеңЁйқўиҜ•е®ҳз”Ёиҝҷз§Қж–№жі•пјҢдј°и®ЎйқўиҜ•е®ҳеҸӘдјҡж‘ҮеӨҙеҗ§ гҖӮдёӢйқўиҜҙдёҖдёӘOпјҲnпјүзҡ„з®—жі•гҖӮ В В В В еңЁдёҠйқўзҡ„зү№пјҲfeiпјүзӮ№пјҲhuaпјүдёӯпјҢж—ўз„¶жңүp<q, йӮЈд№Ҳ2p<2qпјҢйӮЈд№Ҳ вҖңжҲ‘вҖқеңЁеүҚйқўжҜ”дҪ е°Ҹзҡ„ж•°йғҪжІЎиў«йҖүдёҠпјҢдҪ еҗҺйқўз”ҹжҲҗж–°зҡ„дё‘ж•°дёҖе®ҡжҜ”вҖңжҲ‘вҖқеӨ§еҗ§пјҢйӮЈд№ҲдҪ д№ҳ2 з”ҹжҲҗзҡ„дё‘ж•°дёҖе®ҡжҜ”жҲ‘д№ҳ2зҡ„еӨ§еҗ§пјҢйӮЈд№ҲеңЁжҲ‘йҖүдёҠд№ӢеҗҺдҪ жүҚжңүжңәдјҡйҖүдёҠгҖӮ е…¶е®һжҜҸж¬ЎжҲ‘д»¬еҸӘз”ЁжҜ”иҫғ3дёӘж•°пјҡз”ЁдәҺд№ҳ2зҡ„жңҖе°Ҹзҡ„ж•°гҖҒз”ЁдәҺд№ҳ3зҡ„жңҖе°Ҹзҡ„ж•°пјҢз”ЁдәҺд№ҳ5зҡ„жңҖе°Ҹзҡ„ ж•°гҖӮд№ҹе°ұжҳҜжҜ”иҫғ(2x , 3y, 5*z) пјҢx>=y>=zзҡ„пјҢ йҮҚзӮ№иҜҙиҜҙдёӢйқўд»Јз Ғдёӯpзҡ„дҪңз”Ёпјҡint p[] = new int[] { 0, 0, 0 }; p[0]иЎЁзӨәжңҖе°Ҹз”ЁдәҺ д№ҳ2жҜ”иҫғж•°еңЁж•°з»„aдёӯзҡ„гҖҗдҪҚзҪ®гҖ‘гҖӮ
