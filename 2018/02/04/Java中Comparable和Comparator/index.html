JavaдёӯComparableе’ҢComparator

2018/02/04 Java е…ұ 957 еӯ—пјҢзәҰ 3 еҲҶй’ҹ

й“ҫжҺҘ

Comparableз®Җд»Ӣ

гҖҖгҖҖComparableжҳҜжҺ’еәҸжҺҘеҸЈгҖӮиӢҘдёҖдёӘзұ»е®һзҺ°дәҶComparableжҺҘеҸЈпјҢе°ұж„Ҹе‘ізқҖиҜҘзұ»ж”ҜжҢҒжҺ’еәҸгҖӮе®һзҺ°дәҶComparableжҺҘеҸЈзҡ„зұ»зҡ„еҜ№иұЎзҡ„еҲ—иЎЁжҲ–ж•°з»„еҸҜд»ҘйҖҡиҝҮCollections.sortжҲ–Arrays.sortиҝӣиЎҢиҮӘеҠЁжҺ’еәҸгҖӮ

package java.lang;
import java.util.*;
public interface Comparable<T> 
{
    public int compareTo(T o);
}

Comparatorз®Җд»Ӣ

гҖҖгҖҖComparatorжҳҜжҜ”иҫғжҺҘеҸЈпјҢжҲ‘д»¬еҰӮжһңйңҖиҰҒжҺ§еҲ¶жҹҗдёӘзұ»зҡ„ж¬ЎеәҸпјҢиҖҢиҜҘзұ»жң¬иә«дёҚж”ҜжҢҒжҺ’еәҸ(еҚіжІЎжңүе®һзҺ°ComparableжҺҘеҸЈ)пјҢйӮЈд№ҲжҲ‘д»¬е°ұеҸҜд»Ҙе»әз«ӢдёҖдёӘвҖңиҜҘзұ»зҡ„жҜ”иҫғеҷЁвҖқжқҘиҝӣиЎҢжҺ’еәҸпјҢиҝҷдёӘвҖңжҜ”иҫғеҷЁвҖқеҸӘйңҖиҰҒе®һзҺ°ComparatorжҺҘеҸЈеҚіеҸҜгҖӮд№ҹе°ұжҳҜиҜҙпјҢжҲ‘д»¬еҸҜд»ҘйҖҡиҝҮе®һзҺ°ComparatorжқҘж–°е»әдёҖдёӘжҜ”иҫғеҷЁпјҢз„¶еҗҺйҖҡиҝҮиҝҷдёӘжҜ”иҫғеҷЁеҜ№зұ»иҝӣиЎҢжҺ’еәҸгҖӮиҜҘжҺҘеҸЈе®ҡд№үеҰӮдёӢпјҡ

package java.util;
public interface Comparator<T>
 {
    int compare(T o1, T o2);
    boolean equals(Object obj);
 }

жіЁж„Ҹпјҡ1гҖҒиӢҘдёҖдёӘзұ»иҰҒе®һзҺ°ComparatorжҺҘеҸЈпјҡе®ғдёҖе®ҡиҰҒе®һзҺ°compare(T o1, T o2) еҮҪж•°пјҢдҪҶеҸҜд»ҘдёҚе®һзҺ° equals(Object obj) еҮҪж•°гҖӮ

Comparableе’ҢComparatorеҢәеҲ«жҜ”иҫғ

гҖҖгҖҖComparableжҳҜжҺ’еәҸжҺҘеҸЈпјҢиӢҘдёҖдёӘзұ»е®һзҺ°дәҶComparableжҺҘеҸЈпјҢе°ұж„Ҹе‘ізқҖвҖңиҜҘзұ»ж”ҜжҢҒжҺ’еәҸвҖқгҖӮиҖҢComparatorжҳҜжҜ”иҫғеҷЁпјҢжҲ‘д»¬иӢҘйңҖиҰҒжҺ§еҲ¶жҹҗдёӘзұ»зҡ„ж¬ЎеәҸпјҢеҸҜд»Ҙе»әз«ӢдёҖдёӘвҖңиҜҘзұ»зҡ„жҜ”иҫғеҷЁвҖқжқҘиҝӣиЎҢжҺ’еәҸгҖӮ

ComparableзӣёеҪ“дәҺвҖңеҶ…йғЁжҜ”иҫғеҷЁвҖқпјҢиҖҢComparatorзӣёеҪ“дәҺвҖңеӨ–йғЁжҜ”иҫғеҷЁвҖқгҖӮ

гҖҖгҖҖдёӨз§Қж–№жі•еҗ„жңүдјҳеҠЈпјҢ з”ЁComparable з®ҖеҚ•пјҢ еҸӘиҰҒе®һзҺ°Comparable жҺҘеҸЈзҡ„еҜ№иұЎзӣҙжҺҘе°ұжҲҗдёәдёҖдёӘеҸҜд»ҘжҜ”иҫғзҡ„еҜ№иұЎпјҢдҪҶжҳҜйңҖиҰҒдҝ®ж”№жәҗд»Јз ҒгҖӮ з”ЁComparator зҡ„еҘҪеӨ„жҳҜдёҚйңҖиҰҒдҝ®ж”№жәҗд»Јз ҒпјҢ иҖҢжҳҜеҸҰеӨ–е®һзҺ°дёҖдёӘжҜ”иҫғеҷЁпјҢ еҪ“жҹҗдёӘиҮӘе®ҡд№үзҡ„еҜ№иұЎйңҖиҰҒдҪңжҜ”иҫғзҡ„ж—¶еҖҷпјҢжҠҠжҜ”иҫғеҷЁе’ҢеҜ№иұЎдёҖиө·дј йҖ’иҝҮеҺ»е°ұеҸҜд»ҘжҜ”еӨ§е°ҸдәҶпјҢ е№¶дё”еңЁComparator йҮҢйқўз”ЁжҲ·еҸҜд»ҘиҮӘе·ұе®һзҺ°еӨҚжқӮзҡ„еҸҜд»ҘйҖҡз”Ёзҡ„йҖ»иҫ‘пјҢдҪҝе…¶еҸҜд»ҘеҢ№й…ҚдёҖдәӣжҜ”иҫғз®ҖеҚ•зҡ„еҜ№иұЎпјҢйӮЈж ·е°ұеҸҜд»ҘиҠӮзңҒеҫҲеӨҡйҮҚеӨҚеҠіеҠЁдәҶгҖӮ
