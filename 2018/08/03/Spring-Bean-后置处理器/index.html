Spring Bean еҗҺзҪ®еӨ„зҗҶеҷЁ

2018/08/03 Spring е…ұ 1486 еӯ—пјҢзәҰ 5 еҲҶй’ҹ

BeanPostProcessor жҺҘеҸЈе®ҡд№үеӣһи°ғж–№жі•пјҢдҪ еҸҜд»Ҙе®һзҺ°иҜҘж–№жі•жқҘжҸҗдҫӣиҮӘе·ұзҡ„е®һдҫӢеҢ–йҖ»иҫ‘пјҢдҫқиө–и§ЈжһҗйҖ»иҫ‘зӯүгҖӮдҪ д№ҹеҸҜд»ҘеңЁ Spring е®№еҷЁйҖҡиҝҮжҸ’е…ҘдёҖдёӘжҲ–еӨҡдёӘ BeanPostProcessor зҡ„е®һзҺ°жқҘе®ҢжҲҗе®һдҫӢеҢ–пјҢй…ҚзҪ®е’ҢеҲқе§ӢеҢ–дёҖдёӘ bean д№ӢеҗҺе®һзҺ°дёҖдәӣиҮӘе®ҡд№үйҖ»иҫ‘еӣһи°ғж–№жі•гҖӮ

дҪ еҸҜд»Ҙй…ҚзҪ®еӨҡдёӘ BeanPostProcessor жҺҘеҸЈпјҢйҖҡиҝҮи®ҫзҪ® BeanPostProcessor е®һзҺ°зҡ„ Ordered жҺҘеҸЈжҸҗдҫӣзҡ„ order еұһжҖ§жқҘжҺ§еҲ¶иҝҷдәӣ BeanPostProcessor жҺҘеҸЈзҡ„жү§иЎҢйЎәеәҸгҖӮ

иҝҷжҳҜе®һзҺ° BeanPostProcessor зҡ„йқһеёёз®ҖеҚ•зҡ„дҫӢеӯҗпјҢе®ғеңЁд»»дҪ• bean зҡ„еҲқе§ӢеҢ–зҡ„д№ӢеүҚе’Ңд№ӢеҗҺиҫ“е…ҘиҜҘ bean зҡ„еҗҚз§°гҖӮдҪ еҸҜд»ҘеңЁеҲқе§ӢеҢ– bean зҡ„д№ӢеүҚе’Ңд№ӢеҗҺе®һзҺ°жӣҙеӨҚжқӮзҡ„йҖ»иҫ‘пјҢеӣ дёәдҪ жңүдёӨдёӘи®ҝй—®еҶ…зҪ® bean еҜ№иұЎзҡ„еҗҺзҪ®еӨ„зҗҶзЁӢеәҸзҡ„ж–№жі•гҖӮ

XML

public class InitHelloWorld implements BeanPostProcessor {
   public Object postProcessBeforeInitialization(Object bean, String beanName) throws BeansException {
      System.out.println("BeforeInitialization : " + beanName);
      return bean;  // you can return any other object as well
   }
   public Object postProcessAfterInitialization(Object bean, String beanName) throws BeansException {
      System.out.println("AfterInitialization : " + beanName);
      return bean;  // you can return any other object as well
   }
}

<?xml version="1.0" encoding="UTF-8"?>

<beans xmlns="http://www.springframework.org/schema/beans"
    xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
    xsi:schemaLocation="http://www.springframework.org/schema/beans
    http://www.springframework.org/schema/beans/spring-beans-3.0.xsd">

   <bean id="helloWorld" class="com.tutorialspoint.HelloWorld"
       init-method="init" destroy-method="destroy">
       <property name="message" value="Hello World!"/>
   </bean>

   <bean class="com.tutorialspoint.InitHelloWorld" />

</beans>

жіЁи§Ј

дёҖж—ҰдҪ еҲӣе»әжәҗд»Јз Ғе’Ң bean й…ҚзҪ®ж–Үд»¶е®ҢжҲҗеҗҺпјҢжҲ‘д»¬е°ұеҸҜд»ҘиҝҗиЎҢиҜҘеә”з”ЁзЁӢеәҸгҖӮеҰӮжһңдҪ зҡ„еә”з”ЁзЁӢеәҸдёҖеҲҮйғҪжӯЈеёёпјҢе°Ҷиҫ“еҮәд»ҘдёӢдҝЎжҒҜпјҡ

BeforeInitialization : helloWorld
Bean is going through init.
AfterInitialization : helloWorld
Your Message : Hello World!
Bean will destroy now.
