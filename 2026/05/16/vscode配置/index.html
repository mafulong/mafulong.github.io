VSCode 设置指南

配置通过 GitHub 账号同步（Settings Sync）。 Cmd+Shift+P → Settings Sync: Turn On → 登录 GitHub → 选择同步项即可。 同步内容：设置、快捷键、插件、代码片段、UI 状态。

四、gopls 配置备忘（settings.json）

Inlay Hints 正确写法

"gopls": {
  "ui.inlayhint.hints": {           // Hierarchy=ui.inlayhint, Name=hints
    "assignVariableTypes": true,    // 变量类型提示
    "compositeLiteralFields": true, // 结构体字段名提示
    "parameterNames": true          // 调用参数名提示
  }
}

❌ 错误写法（会报 Invalid settings）："ui.inlayHints.X" 或 "hints.X"

避免双重格式化

删除 go.formatTool / go.formatOnSave，由 [go] 块统一走 gopls+gofumpt：

"gopls": { "formatting.gofumpt": true },
"[go]": { "editor.defaultFormatter": "golang.go", "editor.formatOnSave": true }
