124. дәҢеҸүж ‘дёӯзҡ„жңҖеӨ§и·Ҝеҫ„е’Ң

2021/03/04 leetcode е…ұ 803 еӯ—пјҢзәҰ 3 еҲҶй’ҹ

title

problem link

и·Ҝеҫ„ иў«е®ҡд№үдёәдёҖжқЎд»Һж ‘дёӯд»»ж„ҸиҠӮзӮ№еҮәеҸ‘пјҢжІҝзҲ¶иҠӮзӮ№-еӯҗиҠӮзӮ№иҝһжҺҘпјҢиҫҫеҲ°д»»ж„ҸиҠӮзӮ№зҡ„еәҸеҲ—гҖӮеҗҢдёҖдёӘиҠӮзӮ№еңЁдёҖжқЎи·Ҝеҫ„еәҸеҲ—дёӯ иҮіеӨҡеҮәзҺ°дёҖж¬Ў гҖӮиҜҘи·Ҝеҫ„ иҮіе°‘еҢ…еҗ«дёҖдёӘ иҠӮзӮ№пјҢдё”дёҚдёҖе®ҡз»ҸиҝҮж №иҠӮзӮ№гҖӮ

и·Ҝеҫ„е’Ң жҳҜи·Ҝеҫ„дёӯеҗ„иҠӮзӮ№еҖјзҡ„жҖ»е’ҢгҖӮ

з»ҷдҪ дёҖдёӘдәҢеҸүж ‘зҡ„ж №иҠӮзӮ№ root пјҢиҝ”еӣһе…¶ жңҖеӨ§и·Ҝеҫ„е’Ң гҖӮ

зӨәдҫӢ 1пјҡ

иҫ“е…Ҙпјҡroot = [1,2,3]
иҫ“еҮәпјҡ6
и§ЈйҮҠпјҡжңҖдјҳи·Ҝеҫ„жҳҜ 2 -> 1 -> 3 пјҢи·Ҝеҫ„е’Ңдёә 2 + 1 + 3 = 6

зӨәдҫӢ 2пјҡ

иҫ“е…Ҙпјҡroot = [-10,9,20,null,null,15,7]
иҫ“еҮәпјҡ42
и§ЈйҮҠпјҡжңҖдјҳи·Ҝеҫ„жҳҜ 15 -> 20 -> 7 пјҢи·Ҝеҫ„е’Ңдёә 15 + 20 + 7 = 42

жҸҗзӨәпјҡ

  • ж ‘дёӯиҠӮзӮ№ж•°зӣ®иҢғеӣҙжҳҜ [1, 3 * 104]
  • -1000 <= Node.val <= 1000

solution

йўҳзӣ®иҜҙжҳҜ вҖқд»Һд»»ж„ҸиҠӮзӮ№еҮәеҸ‘вҖҰвҖҰ.вҖң зңӢе®ҢиҝҷдёӘжҸҸиҝ°жҲ‘дјҡжғіеҲ°еӨ§жҰӮзҺҮжҳҜиҰҒд№Ҳе…ЁеұҖи®°еҪ•жңҖеӨ§еҖјпјҢиҰҒд№ҲеҸҢйҖ’еҪ’гҖӮ

  • еҰӮжһңдҪҝз”ЁеҸҢйҖ’еҪ’пјҢйӮЈд№ҲеӨҚжқӮеәҰе°ұжҳҜ $O(N^2)$пјҢе®һйҷ…дёҠпјҢеӯҗж ‘зҡ„и·Ҝеҫ„е’Ңи®Ўз®—еҮәжқҘдәҶпјҢеҸҜд»ҘжҺЁеҜјеҮәзҲ¶иҠӮзӮ№зҡ„жңҖеӨ§и·Ҝеҫ„е’ҢпјҢеӣ жӯӨеҰӮжһңдҪҝз”ЁеҸҢйҖ’еҪ’дјҡжңүйҮҚеӨҚи®Ўз®—гҖӮдёҖдёӘеҸҜиЎҢзҡ„ж–№ејҸжҳҜи®°еҝҶеҢ–йҖ’еҪ’гҖӮ
  • еҰӮжһңдҪҝз”Ёе…ЁеұҖи®°еҪ•жңҖеӨ§еҖјпјҢеҸӘйңҖиҰҒеңЁйҖ’еҪ’зҡ„ж—¶еҖҷ return еҪ“еүҚзҡ„дёҖжқЎиҫ№пјҲдёҠйқўжҸҗдәҶдёҚиғҪжӢҗпјүпјҢе№¶еңЁеҮҪж•°еҶ…йғЁи®Ўз®—д»ҘеҪ“еүҚиҠӮзӮ№еҮәеҸ‘зҡ„жңҖеӨ§и·Ҝеҫ„е’ҢпјҢе№¶жӣҙж–°е…ЁеұҖжңҖеӨ§еҖјеҚіеҸҜгҖӮ иҝҷйҮҢзҡ„ж ёеҝғе…¶е®һжҳҜ return иҫғеӨ§зҡ„дёҖжқЎиҫ№пјҢеӣ дёәиҫғе°Ҹзҡ„иҫ№дёҚеҸҜиғҪжҳҜзӯ”жЎҲгҖӮ
class Solution:
    ans = float('-inf')

    def maxPathSum(self, root: TreeNode) -> int:
        def dfs(node):
            if not node:
                return 0
            l = dfs(node.left)
            r = dfs(node.right)
            self.ans = max(self.ans, max(l, 0) + max(r, 0) + node.val)
            return max(l, r, 0) + node.val
        dfs(root)
        return self.ans
