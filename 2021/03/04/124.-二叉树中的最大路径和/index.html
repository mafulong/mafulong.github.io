жбўуЏ«У»┤Тў» РђЮС╗јС╗╗ТёЈУіѓуѓ╣тЄ║тЈЉРђдРђд.Рђю уюІт«їУ┐ЎСИфТЈЈУ┐░ТѕЉС╝џТЃ│тѕ░тцДТдѓујЄТў»УдЂС╣ѕтЁет▒ђУ«░тйЋТюђтцДтђ╝№╝їУдЂС╣ѕтЈїжђњтйњсђѓ

  • тдѓТъюСй┐ућетЈїжђњтйњ№╝їжѓБС╣ѕтцЇТЮѓт║дт░▒Тў» $O(N^2)$№╝їт«ъжЎЁСИі№╝їтГљТаЉуџёУи»тЙётњїУ«Ау«ЌтЄ║ТЮЦС║є№╝їтЈ»С╗ЦТјет»╝тЄ║уѕХУіѓуѓ╣уџёТюђтцДУи»тЙётњї№╝їтЏаТГцтдѓТъюСй┐ућетЈїжђњтйњС╝џТюЅжЄЇтцЇУ«Ау«ЌсђѓСИђСИфтЈ»УАїуџёТќ╣т╝ЈТў»У«░т┐єтїќжђњтйњсђѓ
  • тдѓТъюСй┐ућетЁет▒ђУ«░тйЋТюђтцДтђ╝№╝їтЈфжюђУдЂтюежђњтйњуџёТЌХтђЎ return тйЊтЅЇуџёСИђТЮАУЙ╣№╝ѕСИіжЮбТЈљС║єСИЇУЃйТІљ№╝Ѕ№╝їт╣ХтюетЄйТЋ░тєЁжЃеУ«Ау«ЌС╗ЦтйЊтЅЇУіѓуѓ╣тЄ║тЈЉуџёТюђтцДУи»тЙётњї№╝їт╣ХТЏ┤Тќ░тЁет▒ђТюђтцДтђ╝тЇ│тЈ»сђѓ У┐ЎжЄїуџёТаИт┐ЃтЁХт«ъТў» return УЙЃтцДуџёСИђТЮАУЙ╣№╝їтЏаСИ║УЙЃт░ЈуџёУЙ╣СИЇтЈ»УЃйТў»уГћТАѕсђѓ
class Solution:
    ans = float('-inf')

    def maxPathSum(self, root: TreeNode) -> int:
        def dfs(node):
            if not node:
                return 0
            l = dfs(node.left)
            r = dfs(node.right)
            self.ans = max(self.ans, max(l, 0) + max(r, 0) + node.val)
            return max(l, r, 0) + node.val
        dfs(root)
        return self.ans
