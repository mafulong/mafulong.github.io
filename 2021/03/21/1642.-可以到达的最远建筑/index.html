Тќ╣Т│ЋСИђ№╝џС╝ўтЁѕжўЪтѕЌ + У┤фт┐Ѓ

ТђЮУи»СИју«ЌТ│Ћ

тюеуД╗тіеуџёУ┐ЄуеІСИГ№╝їТѕЉС╗гС╝џжюђУдЂУІЦт╣▓ТгАжюђУдЂСй┐ућеуаќтЮЌТѕќУђЁТб»тГљуџёТЃЁтєхсђѓтЂЄУ«ЙтйЊтЅЇТѕЉС╗гжюђУдЂуД╗тіетѕ░СИІСИђт╗║уГЉуЅЕ№╝їСйєт┐ЁжА╗Сй┐уће 11 ТъХТб»тГљТѕќУђЁ \Delta h╬ћh СИфуаќтЮЌ№╝їжѓБС╣ѕТѕЉС╗гтдѓСйЋТіЅТІЕТў»Сй┐ућеТб»тГљУ┐ўТў»уаќтЮЌтЉб№╝Ъ

ТѕЉС╗гтЈ»С╗ЦућеУ┤фт┐ЃуџёТђЮУи»ТЮЦТЃ│У┐ЎСИфжЌ«жбўсђѓсђїТб»тГљсђЇуЏИтйЊС║јСИђТгАТђДуџёТЌажЎљжЄЈуаќтЮЌ№╝їжѓБС╣ѕТѕЉС╗гСИђт«џТў»ТііТб»тГљућетюетѕђтѕЃСИісђѓС╣Ът░▒Тў»У»┤№╝їтдѓТъюТѕЉС╗гТюЅ ll ТъХТб»тГљ№╝їжѓБС╣ѕТѕЉС╗гС╝џтюе \Delta h╬ћh ТюђтцДуџёжѓБ ll ТгАСй┐ућеТб»тГљ№╝їУђїтюетЅЕСйЎуџёТЃЁтєхСИІСй┐ућеуаќтЮЌсђѓ

У┐ЎТаиСИђТЮЦ№╝їТѕЉС╗гт░▒тЈ»С╗ЦтЙЌтѕ░ТГБуА«уџёу«ЌТ│ЋС║є№╝џТѕЉС╗гСй┐ућеС╝ўтЁѕжўЪтѕЌт«ъТЌХу╗┤ТіцСИЇУХЁУ┐Є ll СИфТюђтцДуџё \Delta h╬ћh№╝їУ┐ЎС║Џт░▒Тў»Сй┐ућеТб»тГљуџётю░Тќ╣сђѓт»╣С║јтЅЕСйЎуџё \Delta h╬ћh№╝їТѕЉС╗гжюђУдЂСй┐ућеуаќтЮЌ№╝їтЏаТГцжюђУдЂт»╣т«ЃС╗гУ┐ЏУАїу┤»тіа№╝їтдѓТъюТЪљСИђТЌХтѕ╗У┐ЎСИфу┤»тіатђ╝УХЁУ┐ЄС║єуаќтЮЌуџёТЋ░уЏ« bb№╝їжѓБС╣ѕТѕЉС╗гт░▒тєЇС╣ЪТЌаТ│ЋуД╗тіеС║єсђѓ


class Solution:
    def furthestBuilding(self, heights: List[int], bricks: int, ladders: int) -> int:
        n = len(heights)
        sumH = 0
        hq = []
        from heapq import heappush, heappop
        for i in range(1, n):
            diff = heights[i] - heights[i - 1]
            if diff <= 0:
                continue
            if ladders > 0:
                ladders -= 1
                heappush(hq, diff)
                continue
            heappush(hq, diff)
            mindiff = heappop(hq)
            sumH += mindiff
            if sumH > bricks:
                return i - 1
        return n - 1
