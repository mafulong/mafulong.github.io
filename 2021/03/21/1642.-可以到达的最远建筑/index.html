1642. еҸҜд»ҘеҲ°иҫҫзҡ„жңҖиҝңе»әзӯ‘

2021/03/21 leetcode е…ұ 1381 еӯ—пјҢзәҰ 4 еҲҶй’ҹ

problem link

з»ҷдҪ дёҖдёӘж•ҙж•°ж•°з»„ heights пјҢиЎЁзӨәе»әзӯ‘зү©зҡ„й«ҳеәҰгҖӮеҸҰжңүдёҖдәӣз –еқ— bricks е’ҢжўҜеӯҗ ladders гҖӮ

дҪ д»Һе»әзӯ‘зү© 0 ејҖе§Ӣж—…зЁӢпјҢдёҚж–ӯеҗ‘еҗҺйқўзҡ„е»әзӯ‘зү©з§»еҠЁпјҢжңҹй—ҙеҸҜиғҪдјҡз”ЁеҲ°з –еқ—жҲ–жўҜеӯҗгҖӮ

еҪ“д»Һе»әзӯ‘зү© i з§»еҠЁеҲ°е»әзӯ‘зү© i+1пјҲдёӢж Ү д»Һ 0 ејҖе§Ӣ пјүж—¶пјҡ

  • еҰӮжһңеҪ“еүҚе»әзӯ‘зү©зҡ„й«ҳеәҰ еӨ§дәҺжҲ–зӯүдәҺ дёӢдёҖе»әзӯ‘зү©зҡ„й«ҳеәҰпјҢеҲҷдёҚйңҖиҰҒжўҜеӯҗжҲ–з –еқ—
  • еҰӮжһңеҪ“еүҚе»әзӯ‘зҡ„й«ҳеәҰ е°ҸдәҺ дёӢдёҖдёӘе»әзӯ‘зҡ„й«ҳеәҰпјҢжӮЁеҸҜд»ҘдҪҝз”Ё дёҖжһ¶жўҜеӯҗ жҲ– (h[i+1] - h[i]) дёӘз –еқ—

еҰӮжһңд»ҘжңҖдҪіж–№ејҸдҪҝз”Ёз»ҷе®ҡзҡ„жўҜеӯҗе’Ңз –еқ—пјҢиҝ”еӣһдҪ еҸҜд»ҘеҲ°иҫҫзҡ„жңҖиҝңе»әзӯ‘зү©зҡ„дёӢж ҮпјҲдёӢж Ү д»Һ 0 ејҖе§Ӣ пјүгҖӮ

зӨәдҫӢ 1пјҡ

иҫ“е…Ҙпјҡheights = [4,2,7,6,9,14,12], bricks = 5, ladders = 1
иҫ“еҮәпјҡ4
и§ЈйҮҠпјҡд»Һе»әзӯ‘зү© 0 еҮәеҸ‘пјҢдҪ еҸҜд»ҘжҢүжӯӨж–№жЎҲе®ҢжҲҗж—…зЁӢпјҡ
- дёҚдҪҝз”Ёз –еқ—жҲ–жўҜеӯҗеҲ°иҫҫе»әзӯ‘зү© 1 пјҢеӣ дёә 4 >= 2
- дҪҝз”Ё 5 дёӘз –еқ—еҲ°иҫҫе»әзӯ‘зү© 2 гҖӮдҪ еҝ…йЎ»дҪҝз”Ёз –еқ—жҲ–жўҜеӯҗпјҢеӣ дёә 2 < 7
- дёҚдҪҝз”Ёз –еқ—жҲ–жўҜеӯҗеҲ°иҫҫе»әзӯ‘зү© 3 пјҢеӣ дёә 7 >= 6
- дҪҝз”Ёе”ҜдёҖзҡ„жўҜеӯҗеҲ°иҫҫе»әзӯ‘зү© 4 гҖӮдҪ еҝ…йЎ»дҪҝз”Ёз –еқ—жҲ–жўҜеӯҗпјҢеӣ дёә 6 < 9
ж— жі•и¶ҠиҝҮе»әзӯ‘зү© 4 пјҢеӣ дёәжІЎжңүжӣҙеӨҡз –еқ—жҲ–жўҜеӯҗгҖӮ

зӨәдҫӢ 2пјҡ

иҫ“е…Ҙпјҡheights = [4,12,2,7,3,18,20,3,19], bricks = 10, ladders = 2
иҫ“еҮәпјҡ7

зӨәдҫӢ 3пјҡ

иҫ“е…Ҙпјҡheights = [14,3,19,3], bricks = 17, ladders = 0
иҫ“еҮәпјҡ3

жҸҗзӨәпјҡ

  • 1 <= heights.length <= 105
  • 1 <= heights[i] <= 106
  • 0 <= bricks <= 109
  • 0 <= ladders <= heights.length

solution

ж–№жі•дёҖпјҡдјҳе…ҲйҳҹеҲ— + иҙӘеҝғ

жҖқи·ҜдёҺз®—жі•

еңЁз§»еҠЁзҡ„иҝҮзЁӢдёӯпјҢжҲ‘д»¬дјҡйңҖиҰҒиӢҘе№Іж¬ЎйңҖиҰҒдҪҝз”Ёз –еқ—жҲ–иҖ…жўҜеӯҗзҡ„жғ…еҶөгҖӮеҒҮи®ҫеҪ“еүҚжҲ‘д»¬йңҖиҰҒз§»еҠЁеҲ°дёӢдёҖе»әзӯ‘зү©пјҢдҪҶеҝ…йЎ»дҪҝз”Ё 11 жһ¶жўҜеӯҗжҲ–иҖ… \Delta hО”h дёӘз –еқ—пјҢйӮЈд№ҲжҲ‘д»¬еҰӮдҪ•жҠүжӢ©жҳҜдҪҝз”ЁжўҜеӯҗиҝҳжҳҜз –еқ—е‘ўпјҹ

жҲ‘д»¬еҸҜд»Ҙз”ЁиҙӘеҝғзҡ„жҖқи·ҜжқҘжғіиҝҷдёӘй—®йўҳгҖӮгҖҢжўҜеӯҗгҖҚзӣёеҪ“дәҺдёҖж¬ЎжҖ§зҡ„ж— йҷҗйҮҸз –еқ—пјҢйӮЈд№ҲжҲ‘д»¬дёҖе®ҡжҳҜжҠҠжўҜеӯҗз”ЁеңЁеҲҖеҲғдёҠгҖӮд№ҹе°ұжҳҜиҜҙпјҢеҰӮжһңжҲ‘д»¬жңү ll жһ¶жўҜеӯҗпјҢйӮЈд№ҲжҲ‘д»¬дјҡеңЁ \Delta hО”h жңҖеӨ§зҡ„йӮЈ ll ж¬ЎдҪҝз”ЁжўҜеӯҗпјҢиҖҢеңЁеү©дҪҷзҡ„жғ…еҶөдёӢдҪҝз”Ёз –еқ—гҖӮ

иҝҷж ·дёҖжқҘпјҢжҲ‘д»¬е°ұеҸҜд»Ҙеҫ—еҲ°жӯЈзЎ®зҡ„з®—жі•дәҶпјҡжҲ‘д»¬дҪҝз”Ёдјҳе…ҲйҳҹеҲ—е®һж—¶з»ҙжҠӨдёҚи¶…иҝҮ ll дёӘжңҖеӨ§зҡ„ \Delta hО”hпјҢиҝҷдәӣе°ұжҳҜдҪҝз”ЁжўҜеӯҗзҡ„ең°ж–№гҖӮеҜ№дәҺеү©дҪҷзҡ„ \Delta hО”hпјҢжҲ‘д»¬йңҖиҰҒдҪҝз”Ёз –еқ—пјҢеӣ жӯӨйңҖиҰҒеҜ№е®ғд»¬иҝӣиЎҢзҙҜеҠ пјҢеҰӮжһңжҹҗдёҖж—¶еҲ»иҝҷдёӘзҙҜеҠ еҖји¶…иҝҮдәҶз –еқ—зҡ„ж•°зӣ® bbпјҢйӮЈд№ҲжҲ‘д»¬е°ұеҶҚд№ҹж— жі•з§»еҠЁдәҶгҖӮ


class Solution:
    def furthestBuilding(self, heights: List[int], bricks: int, ladders: int) -> int:
        n = len(heights)
        sumH = 0
        hq = []
        from heapq import heappush, heappop
        for i in range(1, n):
            diff = heights[i] - heights[i - 1]
            if diff <= 0:
                continue
            if ladders > 0:
                ladders -= 1
                heappush(hq, diff)
                continue
            heappush(hq, diff)
            mindiff = heappop(hq)
            sumH += mindiff
            if sumH > bricks:
                return i - 1
        return n - 1
