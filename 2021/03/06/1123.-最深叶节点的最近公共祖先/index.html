1123. жңҖж·ұеҸ¶иҠӮзӮ№зҡ„жңҖиҝ‘е…¬е…ұзҘ–е…Ҳ

2021/03/06 leetcode е…ұ 784 еӯ—пјҢзәҰ 3 еҲҶй’ҹ

title

problem link

з»ҷдҪ дёҖдёӘжңүж №иҠӮзӮ№зҡ„дәҢеҸүж ‘пјҢжүҫеҲ°е®ғжңҖж·ұзҡ„еҸ¶иҠӮзӮ№зҡ„жңҖиҝ‘е…¬е…ұзҘ–е…ҲгҖӮ

еӣһжғідёҖдёӢпјҡ

  • еҸ¶иҠӮзӮ№ жҳҜдәҢеҸүж ‘дёӯжІЎжңүеӯҗиҠӮзӮ№зҡ„иҠӮзӮ№
  • ж ‘зҡ„ж №иҠӮзӮ№зҡ„ ж·ұеәҰ дёә 0пјҢеҰӮжһңжҹҗдёҖиҠӮзӮ№зҡ„ж·ұеәҰдёә dпјҢйӮЈе®ғзҡ„еӯҗиҠӮзӮ№зҡ„ж·ұеәҰе°ұжҳҜ d+1
  • еҰӮжһңжҲ‘д»¬еҒҮе®ҡ A жҳҜдёҖз»„иҠӮзӮ№ S зҡ„ жңҖиҝ‘е…¬е…ұзҘ–е…ҲпјҢS дёӯзҡ„жҜҸдёӘиҠӮзӮ№йғҪеңЁд»Ҙ A дёәж №иҠӮзӮ№зҡ„еӯҗж ‘дёӯпјҢдё” A зҡ„ж·ұеәҰиҫҫеҲ°жӯӨжқЎд»¶дёӢеҸҜиғҪзҡ„жңҖеӨ§еҖјгҖӮ

жіЁж„Ҹпјҡжң¬йўҳдёҺеҠӣжүЈ 865 йҮҚеӨҚпјҡhttps://leetcode-cn.com/problems/smallest-subtree-with-all-the-deepest-nodes/

зӨәдҫӢ 1пјҡ

иҫ“е…Ҙпјҡroot = [3,5,1,6,2,0,8,null,null,7,4]
иҫ“еҮәпјҡ[2,7,4]
и§ЈйҮҠпјҡ
жҲ‘д»¬иҝ”еӣһеҖјдёә 2 зҡ„иҠӮзӮ№пјҢеңЁеӣҫдёӯз”Ёй»„иүІж Үи®°гҖӮ
еңЁеӣҫдёӯз”Ёи“қиүІж Үи®°зҡ„жҳҜж ‘зҡ„жңҖж·ұзҡ„иҠӮзӮ№гҖӮ
жіЁж„ҸпјҢиҠӮзӮ№ 6гҖҒ0 е’Ң 8 д№ҹжҳҜеҸ¶иҠӮзӮ№пјҢдҪҶжҳҜе®ғд»¬зҡ„ж·ұеәҰжҳҜ 2 пјҢиҖҢиҠӮзӮ№ 7 е’Ң 4 зҡ„ж·ұеәҰжҳҜ 3 гҖӮ

зӨәдҫӢ 2пјҡ

иҫ“е…Ҙпјҡroot = [1]
иҫ“еҮәпјҡ[1]
и§ЈйҮҠпјҡж №иҠӮзӮ№жҳҜж ‘дёӯжңҖж·ұзҡ„иҠӮзӮ№пјҢе®ғжҳҜе®ғжң¬иә«зҡ„жңҖиҝ‘е…¬е…ұзҘ–е…ҲгҖӮ

зӨәдҫӢ 3пјҡ

иҫ“е…Ҙпјҡroot = [0,1,3,null,2]
иҫ“еҮәпјҡ[2]
и§ЈйҮҠпјҡж ‘дёӯжңҖж·ұзҡ„еҸ¶иҠӮзӮ№жҳҜ 2 пјҢжңҖиҝ‘е…¬е…ұзҘ–е…ҲжҳҜе®ғиҮӘе·ұгҖӮ

жҸҗзӨәпјҡ

  • з»ҷдҪ зҡ„ж ‘дёӯе°Ҷжңү 1 еҲ° 1000 дёӘиҠӮзӮ№гҖӮ
  • ж ‘дёӯжҜҸдёӘиҠӮзӮ№зҡ„еҖјйғҪеңЁ 1 еҲ° 1000 д№Ӣй—ҙгҖӮ
  • жҜҸдёӘиҠӮзӮ№зҡ„еҖјйғҪжҳҜзӢ¬дёҖж— дәҢзҡ„гҖӮ

solution

class Solution:
    def lcaDeepestLeaves(self, root: TreeNode) -> TreeNode:
        def dfs(node):
            if not node: return (None, 0)
            l, ld = dfs(node.left)
            r, rd = dfs(node.right)
            if ld == rd:
                return (node, ld + 1)
            elif ld > rd:
                return (l, ld + 1)
            else:
                return (r, rd + 1)
        return dfs(root)[0]
