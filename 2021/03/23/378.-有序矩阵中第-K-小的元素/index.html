378. 有序矩阵中第 K 小的元素

2021/03/23 leetcode 共 688 字，约 2 分钟

title

problem link

给你一个 n x n 矩阵 matrix ，其中每行和每列元素均按升序排序，找到矩阵中第 k 小的元素。 请注意，它是 排序后 的第 k 小元素，而不是第 k 个 不同 的元素。

示例 1：

输入：matrix = [[1,5,9],[10,11,13],[12,13,15]], k = 8
输出：13
解释：矩阵中的元素为 [1,5,9,10,11,12,13,13,15]，第 8 小元素是 13

示例 2：

输入：matrix = [[-5]], k = 1
输出：-5

提示：

  • n == matrix.length
  • n == matrix[i].length
  • 1 <= n <= 300
  • -109 <= matrix[i][j] <= -109
  • 题目数据 保证 matrix 中的所有行和列都按 非递减顺序 排列
  • 1 <= k <= n2

solution

class Solution:
    def kthSmallest(self, matrix: List[List[int]], k: int) -> int:
        if not matrix:
            return 0
        m, n = len(matrix), len(matrix[0])

        def check(x) -> bool:
            w = 0
            i, j = m - 1, 0
            while i >= 0:
                while j < n and matrix[i][j] <= x:
                    j += 1
                w += j
                i -= 1
                if w >= k:
                    return False
            return w < k

        l, r = matrix[0][0], matrix[-1][-1]
        while l <= r:
            mid = (l + r) // 2
            if check(mid):
                l = mid + 1
            else:
                r = mid - 1
        return l
