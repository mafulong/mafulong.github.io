1835. жүҖжңүж•°еҜ№жҢүдҪҚдёҺз»“жһңзҡ„ејӮжҲ–е’Ң

2021/04/24 leetcode е…ұ 758 еӯ—пјҢзәҰ 3 еҲҶй’ҹ

title

problem link

еҲ—иЎЁзҡ„ ејӮжҲ–е’ҢпјҲXOR sumпјүжҢҮеҜ№жүҖжңүе…ғзҙ иҝӣиЎҢжҢүдҪҚ XOR иҝҗз®—зҡ„з»“жһңгҖӮеҰӮжһңеҲ—иЎЁдёӯд»…жңүдёҖдёӘе…ғзҙ пјҢйӮЈд№Ҳе…¶ ејӮжҲ–е’Ң е°ұзӯүдәҺиҜҘе…ғзҙ гҖӮ

  • дҫӢеҰӮпјҢ[1,2,3,4] зҡ„ ејӮжҲ–е’Ң зӯүдәҺ 1 XOR 2 XOR 3 XOR 4 = 4 пјҢиҖҢ [3] зҡ„ ејӮжҲ–е’Ң зӯүдәҺ 3 гҖӮ

з»ҷдҪ дёӨдёӘдёӢж Ү д»Һ 0 ејҖе§Ӣ и®Ўж•°зҡ„ж•°з»„ arr1 е’Ң arr2 пјҢдёӨж•°з»„еқҮз”ұйқһиҙҹж•ҙж•°з»„жҲҗгҖӮ

ж №жҚ®жҜҸдёӘ (i, j) ж•°еҜ№пјҢжһ„йҖ дёҖдёӘз”ұ arr1[i] AND arr2[j]пјҲжҢүдҪҚ AND иҝҗз®—пјүз»“жһңз»„жҲҗзҡ„еҲ—иЎЁгҖӮе…¶дёӯ 0 <= i < arr1.length дё” 0 <= j < arr2.length гҖӮ

иҝ”еӣһдёҠиҝ°еҲ—иЎЁзҡ„ ејӮжҲ–е’Ң гҖӮ

зӨәдҫӢ 1пјҡ

иҫ“е…Ҙпјҡarr1 = [1,2,3], arr2 = [6,5]
иҫ“еҮәпјҡ0
и§ЈйҮҠпјҡеҲ—иЎЁ = [1 AND 6, 1 AND 5, 2 AND 6, 2 AND 5, 3 AND 6, 3 AND 5] = [0,1,2,0,2,1] пјҢ
ејӮжҲ–е’Ң = 0 XOR 1 XOR 2 XOR 0 XOR 2 XOR 1 = 0 гҖӮ

зӨәдҫӢ 2пјҡ

иҫ“е…Ҙпјҡarr1 = [12], arr2 = [4]
иҫ“еҮәпјҡ4
и§ЈйҮҠпјҡеҲ—иЎЁ = [12 AND 4] = [4] пјҢејӮжҲ–е’Ң = 4 гҖӮ

жҸҗзӨәпјҡ

  • 1 <= arr1.length, arr2.length <= 105
  • 0 <= arr1[i], arr2[j] <= 109

solution

еҸӮиҖғ

class Solution:
    def getXORSum(self, arr1: List[int], arr2: List[int]) -> int:
        ans = 0
        for k in range(30, -1, -1):
            cnt1 = sum([1 for l in arr1 if (l & (1 << k)) ])
            cnt2 = sum([1 for l in arr2 if (l & (1 << k)) ])
            if cnt1 % 2 ==1 and cnt2 % 2==1:
                ans |= (1<<k)
        return ans
