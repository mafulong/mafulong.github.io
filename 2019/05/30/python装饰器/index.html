pythonиЈ…йҘ°еҷЁ

pythonиЈ…йҘ°еҷЁжң¬иҙЁдёҠе°ұжҳҜдёҖдёӘеҮҪж•°пјҢе®ғеҸҜд»Ҙи®©е…¶д»–еҮҪж•°еңЁдёҚйңҖиҰҒеҒҡд»»дҪ•д»Јз ҒеҸҳеҠЁзҡ„еүҚжҸҗдёӢеўһеҠ йўқеӨ–зҡ„еҠҹиғҪпјҢиЈ…йҘ°еҷЁзҡ„иҝ”еӣһеҖјд№ҹжҳҜдёҖдёӘеҮҪж•°еҜ№иұЎпјҲеҮҪж•°зҡ„жҢҮй’Ҳпјү

pythonиЈ…йҘ°еҷЁжңүеҫҲеӨҡз»Ҹе…ёзҡ„еә”з”ЁеңәжҷҜпјҢжҜ”еҰӮпјҡжҸ’е…Ҙж—Ҙеҝ—гҖҒжҖ§иғҪжөӢиҜ•гҖҒдәӢеҠЎеӨ„зҗҶгҖҒжқғйҷҗж ЎйӘҢзӯүгҖӮиЈ…йҘ°еҷЁжҳҜи§ЈеҶіиҝҷзұ»й—®йўҳзҡ„з»қдҪіи®ҫи®ЎгҖӮе№¶дё”д»Һеј•е…Ҙдёӯзҡ„еҲ—еӯҗдёӯжҲ‘д»¬д№ҹеҸҜд»ҘеҪ’зәіеҮәпјҡиЈ…йҘ°еҷЁжңҖеӨ§зҡ„дҪңз”Ёе°ұжҳҜеҜ№дәҺжҲ‘д»¬е·Із»ҸеҶҷеҘҪзҡ„зЁӢеәҸпјҢжҲ‘д»¬еҸҜд»ҘжҠҪзҰ»еҮәдёҖдәӣйӣ·еҗҢзҡ„д»Јз Ғз»„е»әеӨҡдёӘзү№е®ҡеҠҹиғҪзҡ„иЈ…йҘ°еҷЁпјҢиҝҷж ·жҲ‘д»¬е°ұеҸҜд»Ҙй’ҲеҜ№дёҚеҗҢзҡ„йңҖжұӮеҺ»дҪҝз”Ёзү№е®ҡзҡ„иЈ…йҘ°еҷЁпјҢиҝҷж—¶еӣ дёәжәҗз ҒеҺ»йҷӨдәҶеӨ§йҮҸжіӣеҢ–зҡ„еҶ…е®№иҖҢдҪҝеҫ—жәҗз Ғе…·жңүжӣҙеҠ жё…жҷ°зҡ„йҖ»иҫ‘гҖӮ

еҮҪж•°иЈ…йҘ°еҷЁ

жҲ‘д»¬иҝҳжҳҜд»ҘдёәеҮҪж•°ж·»еҠ и®Ўж—¶еҠҹиғҪдёәдҫӢпјҢи®Іиҝ°еҮҪж•°иЈ…йҘ°еҷЁгҖӮ

import time

def decorator(func):
    def wrapper(*args, **kwargs):
        start_time = time.time()
        func()
        end_time = time.time()
        print(end_time - start_time)

    return wrapper

@decorator 
def func():
    time.sleep(0.8)

func() # еҮҪж•°и°ғз”Ё
# иҫ“еҮәпјҡ0.800644397735595

еңЁдёҠйқўд»Јз Ғдёӯ funcжҳҜжҲ‘иҰҒиЈ…йҘ°еҷЁзҡ„еҮҪж•°пјҢжҲ‘жғіз”ЁиЈ…йҘ°еҷЁжҳҫзӨәfuncеҮҪж•°иҝҗиЎҢзҡ„ж—¶й—ҙгҖӮ@decoratorиҝҷдёӘиҜӯжі•зӣёеҪ“дәҺ жү§иЎҢ func = decorator(func)пјҢдёәfuncеҮҪж•°иЈ…йҘ°е№¶иҝ”еӣһгҖӮеңЁжқҘзңӢдёҖдёӢжҲ‘д»¬зҡ„иЈ…йҘ°еҷЁеҮҪж•° - decoratorпјҢиҜҘеҮҪж•°зҡ„дј е…ҘеҸӮж•°жҳҜfunc пјҲиў«иЈ…йҘ°еҮҪж•°пјүпјҢиҝ”еӣһеҸӮж•°жҳҜеҶ…еұӮеҮҪж•°гҖӮиҝҷйҮҢзҡ„еҶ…еұӮеҮҪж•°-wrapperпјҢе…¶е®һе°ұзӣёеҪ“дәҺй—ӯеҢ…еҮҪж•°пјҢе®ғиө·еҲ°иЈ…йҘ°з»ҷе®ҡеҮҪж•°зҡ„дҪңз”ЁпјҢwrapperеҸӮж•°дёә*args, kwargsгҖӮ*argsиЎЁзӨәзҡ„еҸӮж•°д»ҘеҲ—иЎЁзҡ„еҪўејҸдј е…ҘпјӣkwargsиЎЁзӨәзҡ„еҸӮж•°д»Ҙеӯ—е…ёзҡ„еҪўејҸдј е…Ҙпјҡ

зұ»иЈ…йҘ°еҷЁ

еүҚйқўжҲ‘д»¬жҸҗеҲ°зҡ„йғҪжҳҜи®© еҮҪж•°дҪңдёәиЈ…йҘ°еҷЁеҺ»иЈ…йҘ°е…¶д»–зҡ„еҮҪж•°жҲ–иҖ…ж–№жі•пјҢйӮЈд№ҲеҸҜдёҚеҸҜд»Ҙи®© дёҖдёӘзұ»еҸ‘жҢҘиЈ…йҘ°еҷЁзҡ„дҪңз”Ёе‘ўпјҹзӯ”жЎҲиӮҜе®ҡжҳҜеҸҜд»Ҙзҡ„пјҢдёҖеҲҮзҡҶеҜ№иұЎеҡ’пјҢеҮҪж•°е’Ңзұ»жң¬иҙЁжІЎжңүд»Җд№ҲдёҚдёҖж ·гҖӮ

class Decorator(object):
    def __init__(self, f):
        self.f = f
    def __call__(self):
        print("decorator start")
        self.f()
        print("decorator end")

@Decorator
def func():
    print("func")

func()

иҝҷйҮҢжңүжіЁж„Ҹзҡ„жҳҜпјҡcall()жҳҜдёҖдёӘзү№ж®Ҡж–№жі•пјҢе®ғеҸҜе°ҶдёҖдёӘзұ»е®һдҫӢеҸҳжҲҗдёҖдёӘеҸҜи°ғз”ЁеҜ№иұЎ

иЈ…йҘ°еҷЁй“ҫ

дёҖдёӘpythonеҮҪж•°д№ҹеҸҜд»Ҙиў«еӨҡдёӘиЈ…йҘ°еҷЁдҝ®йҘ°

жҢүз…§ж¬ЎеәҸдҫқж¬Ўжү§иЎҢпјҢд»ҺдёҠеҲ°дёӢ

functools

def decorator(func):
    def inner_function():
        pass
    return inner_function

@decorator
def func():
    pass

print(func.__name__)

# иҫ“еҮәпјҡ inner_function

дёҠиҝ°д»Јз ҒжңҖеҗҺжү§иЎҢзҡ„з»“жһңдёҚжҳҜ funcпјҢиҖҢжҳҜ inner_functionпјҒиҝҷиЎЁзӨәиў«иЈ…йҘ°еҮҪж•°иҮӘиә«зҡ„дҝЎжҒҜдёўеӨұдәҶпјҒжҖҺд№ҲжүҚиғҪйҒҝе…Қиҝҷз§Қй—®йўҳзҡ„еҸ‘з”ҹе‘ўпјҹ

еҸҜд»ҘеҖҹеҠ©functools.wraps()еҮҪж•°пјҡ

from functools import wraps
def decorator(func):
    @wraps(func) 
    def inner_function():
        pass
    return inner_function

@decorator
def func():
    pass

print(func.__name__)

#иҫ“еҮәпјҡ func
