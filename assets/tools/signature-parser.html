EIP-191 (personal_sign)
EIP-712 (typed data)
EIP-191 签名格式: \x19Ethereum Signed Message:\n + 消息长度 + 消息内容
输入签名（65字节 hex: r[32] + s[32] + v[1]）和原始消息文本，即可恢复签名者地址。
EIP-712 签名格式: \x19\x01 + domainSeparator(32 bytes) + hashStruct(message, domain)(32 bytes)
输入完整的 EIP-712 签名数据（64字节），解析 domain separator 和 message hash。
